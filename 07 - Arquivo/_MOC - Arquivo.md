# 🗄️ MOC — Arquivo

Projetos concluídos, áreas abandonadas, roteiros publicados. **Nada é deletado** — apenas movido para cá.

## Por que arquivar (e não apagar)
- Histórico ajuda a reaproveitar pesquisa antiga.
- Wikilinks continuam funcionando.
- Limpa as visões dos MOCs ativos sem perder dado.

## Como arquivar
1. Atualizar frontmatter: `status: arquivado`, `arquivado: YYYY-MM-DD`.
2. Mover arquivo para `07 - Arquivo/`.
3. Remover de MOCs ativos (opcional).
