# Referência — AlphaEvolve: Status Open Source (2026)

## Resumo

AlphaEvolve é um agente de codificação evolucionária da Google DeepMind, baseado no Gemini. O sistema original **não foi liberado como open source** pelo Google — permanece como tecnologia interna. Entretanto, a publicação do paper (arXiv 2506.13131) levou a comunidade a criar várias implementações independentes.

## Implementações Open Source da Comunidade

- **OpenEvolve** (Hugging Face / jamesahou): implementação mais popular, com suporte a modelos via API
  - https://github.com/jamesahou/openevolve
- **OpenAlpha_Evolve** (shyamsaktawat): framework Python com foco em agente autônomo de código
  - https://github.com/shyamsaktawat/OpenAlpha_Evolve
- **CodeEvolve**: agente evolucionário com LLMs + algoritmos genéticos — publicado no arXiv (2510.14150)

## Paper Original

- arXiv: https://arxiv.org/abs/2506.13131 — "AlphaEvolve: A coding agent for scientific and algorithmic discovery"

## Aplicações Reais (Google interno)

- Recuperação de 0,7% da capacidade compute global do Google
- Melhoria na correção de erros em sequenciamento de DNA
- Precisão em previsões de desastres
- Simulações moleculares complexas e neurociência

## Conclusão

AlphaEvolve **não é open source** (Google não publicou o código). Mas o paper é público e há múltiplas reimplementações funcionais da comunidade.
