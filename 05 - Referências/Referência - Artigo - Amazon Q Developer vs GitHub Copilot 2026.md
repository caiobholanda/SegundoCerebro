# Referência — Amazon Q Developer vs GitHub Copilot (2026)

## Comparação de Preços

| Plano | Amazon Q Developer | GitHub Copilot |
|-------|-------------------|----------------|
| Gratuito | Ilimitado + scans de segurança | 2.000 completions + 50 chats/mês |
| Individual pago | Sem plano individual dedicado | $10-39/mês |
| Empresarial | $19/usuário/mês | $19-39/usuário/mês |
| Sobrecarga | Sem custos extras | $0,04 por requisição extra |

## Funcionalidades exclusivas do Amazon Q para AWS

1. **Segurança integrada**: SAST scanning em 12+ linguagens, análise IaC para CloudFormation/CDK, detecção de secrets
2. **Agentes de modernização**: Transforma automaticamente Java 8/11→17/21 e .NET Framework→.NET 8
3. **Integração com Console AWS**: Chat widget nativo no Management Console e AWS CLI
4. **Análise de Infraestrutura**: Acesso a logs CloudWatch e sugestões de correção em tempo real

## Cálculo de custo para equipes

- 50 devs no Amazon Q Pro: $950/mês (inclui security scanning)
- 50 devs no GitHub Copilot Enterprise: $3.000/mês ($60/usuário incluindo GitHub Enterprise Cloud)
- Economia em equipes AWS-first: ~68% de redução de custo

## Veredicto

- **Escolha Amazon Q** se: stack primariamente AWS, necessidade de security scanning built-in, modernização de Java/.NET em larga escala, compliance em indústrias reguladas
- **Escolha GitHub Copilot** se: múltiplas linguagens e clouds, melhor qualidade geral de completions, modo agent totalmente autônomo, workflows de PR no GitHub
