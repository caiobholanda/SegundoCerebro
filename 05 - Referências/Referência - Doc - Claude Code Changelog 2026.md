## 💡 Trechos-chave
> Agent View (Research Preview): painel único listando todas as sessões Claude Code — em execução, aguardando resposta ou concluídas.
> /goal command: define condição de conclusão e Claude trabalha autonomamente até atingir a meta.
> Rate limits doubled for Pro/Max/Team — peak-hour throttling removido.

## 🪞 Reflexão
