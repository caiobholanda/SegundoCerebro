# Referência — Salários AI Engineer Junior Brasil (Glassdoor 2026)

## Resumo
Dados de salário do Glassdoor para cargos de AI Engineer e Junior ML Engineer no Brasil, coletados em março de 2026.

## Dados Principais
- **Junior AI Engineer**: faixa de R$ 3.599 – R$ 4.000/mês (dados Glassdoor com possível subnotificação)
- **Junior Machine Learning Engineer**: média de R$ 6.881/mês; faixa típica R$ 5.255 – R$ 8.500/mês (P25–P75)
- **AI Engineer (geral, todos os níveis)**: R$ 19.500 – R$ 27.000/mês dependendo da senioridade

## Observação
Os dados para "Junior AI Engineer" no Glassdoor BR podem estar subnotificados (cargo muito recente, poucos registros). Os dados de Junior ML Engineer são mais robustos e servem como proxy.
