# Referência - Doc - Claude Agent SDK Overview

## 🔗 Fonte
- **URL**: https://platform.claude.com/docs/en/agent-sdk/overview
- **Autor**: Anthropic
- **Data publicação**: 2026 (consultado 2026-05-13)

## 📝 Resumo (suas palavras)
O Claude Code SDK foi renomeado para Claude Agent SDK. Permite construir agentes de IA autônomos que leem arquivos, executam comandos, buscam na web e editam código. Suporta MCP (Model Context Protocol) para integração com sistemas externos como bancos de dados, browsers e APIs.

## 💡 Trechos-chave
> Everything that makes Claude Code powerful is available in the SDK — agents can read files, run commands, search codebases out of the box.
> Supports MCP for databases, browsers, APIs, and hundreds more external systems.

## 🪞 Reflexão
Renomeação de Claude Code SDK para Agent SDK sinaliza maturidade da plataforma agentic. Importante para projetos que usam automação — o SDK é a base para o segundo cérebro autônomo.
