# Referência — Claude Mythos & Project Glasswing (2026)

## Claude Mythos Preview

- **Anunciado**: 7 de abril de 2026
- **Posição**: um tier completo acima do Claude Opus 4.7 — modelo mais capaz da Anthropic
- **Destaque**: excepcionalmente capaz em tarefas de segurança computacional
- **Preço** (acesso limitado): $25/M tokens input · $125/M tokens output
- **Disponibilidade**: apenas via Project Glasswing — **não disponível ao público geral**

## Project Glasswing

Iniciativa controlada para usar o Claude Mythos Preview em segurança de software crítico. Parceiros de lançamento:

- Amazon Web Services, Apple, Broadcom, Cisco, CrowdStrike, Google, JPMorganChase, Linux Foundation, Microsoft, NVIDIA, Palo Alto Networks

Acesso estendido a +40 organizações que constroem ou mantêm infraestrutura de software crítico.

## Resultados até maio/2026

- Identificação autônoma de **milhares de vulnerabilidades zero-day** em todos os principais sistemas operacionais e navegadores
- Exploração autônoma de uma vulnerabilidade de RCE (Remote Code Execution) de 17 anos no FreeBSD
- Parceria com o governo dos EUA (NIST/CAISI) para avaliação de modelos de fronteira antes do lançamento
