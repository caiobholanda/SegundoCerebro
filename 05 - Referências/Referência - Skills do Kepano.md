## 📝 Resumo (suas palavras)
Conjunto oficial de 5 Agent Skills que ensinam qualquer agente compatível (Claude Code, Codex CLI, OpenCode) a operar arquivos do Obsidian sem quebrar formato.

## 💡 Skills disponíveis
1. **obsidian-markdown** — wikilinks, embeds, callouts, properties (frontmatter).
2. **obsidian-bases** — arquivos `.base` com views, filtros, fórmulas.
3. **json-canvas** — arquivos `.canvas` com nodes, edges, groups.
4. **obsidian-cli** — operar vault via Obsidian CLI.
5. **defuddle** — extrair markdown limpo de páginas web.

## 🪞 Reflexão
Sem essas skills, o agente cria arquivos que parecem markdown mas falham em wikilinks, frontmatter e canvas. Com elas, tudo "encaixa" no Obsidian nativamente.
