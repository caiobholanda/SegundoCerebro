# Referência — AWS Kiro: IDE Agêntico 2026

## O que é

Kiro é um IDE agêntico desenvolvido pela AWS, construído sobre Code OSS (base do VS Code). Foca em **Spec-Driven Development**: o agente transforma prompts em especificações detalhadas → depois em código, docs e testes.

Não é um serviço AWS tradicional — é uma ferramenta standalone com marca própria "Kiro".

## Disponibilidade e acesso

- **Status**: Public Preview (gratuito com limites)
- **Requer conta AWS?**: Não. Aceita login via Google, GitHub, AWS Builder ID ou AWS IAM Identity Center
- **Como acessar**:
  1. Acesse https://kiro.dev/
  2. Clique em Download (Windows/macOS Intel/macOS Apple Silicon/Linux)
  3. Faça login com Google ou GitHub — sem cartão de crédito, sem waitlist
- **Versão web**: disponível em https://app.kiro.dev/ (preview: apenas região us-east-1)
- **GovCloud**: disponível nas regiões US-East e US-West

## Funcionalidades principais (2026)

- **Spec-Driven Development**: cria spec antes de escrever código
- **Parallel Task Execution**: tarefas independentes rodando em paralelo — redução de ~75% no tempo de implementação de projetos grandes
- **Quick Plan mode**: pula aprovação passo-a-passo para features bem definidas
- **Spec Check**: verificação matemática de requisitos antes que o agente comece a codar (feature lançada em 2026 para combater "AI slop")
- **Modelos suportados**: Claude Sonnet 4.0 e 3.7

## Preços (durante preview)
- Preview: gratuito com limites de uso
- Plano pago futuro: ~$20/mês (confirmado em comparativos com Cursor)
