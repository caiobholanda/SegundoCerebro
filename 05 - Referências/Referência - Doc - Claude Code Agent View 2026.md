# Referência - Doc - Claude Code Agent View 2026

## 🔗 Fonte
- **URL**: https://claude.com/blog/agent-view-in-claude-code
- **Autor**: Anthropic
- **Data publicação**: 2026-05-11

## 📝 Resumo (suas palavras)
Post oficial do lançamento do Agent View como Research Preview em 11/05/2026. Dashboard unificado que lista todas as sessões Claude Code (em execução, aguardando input, concluídas) em uma única interface. Acessível via `claude agents` no CLI.

## 💡 Trechos-chave
> Agent view is available starting today as a Research Preview for anyone on Pro, Max, Team, Enterprise plan or with Claude API access.
> Research preview means the keybindings, command names, and UI may shift before general availability.
> The underlying capability — one list, every session — is the part Anthropic is committing to.

### Funcionalidades
- Lista única de todas as sessões: running, blocked on you, done
- Disparar novas tarefas a partir da view
- Responder a prompts bloqueados
- Parar sessões em execução
- Limpar sessões concluídas
- Acessível via `claude agents`

### Status e disponibilidade
- Lançado: 11/05/2026 (Research Preview)
- Planos: Pro, Max, Team, Enterprise, Claude API
- GA: **sem data anunciada** (UI/keybindings podem mudar antes de GA)

## 🪞 Reflexão
"Research Preview" indica que o produto está funcional mas a interface ainda pode mudar. Anthropic comprometeu-se com o conceito (uma lista, todas as sessões) mas não com a UI específica. Nenhuma data de GA foi anunciada publicamente até 14/05/2026.
