# Referência - Vídeo - Claude Code Transformou Obsidian em Segundo Cérebro

## 🔗 Fonte
- **URL**: https://www.youtube.com/watch?v=GWwGmUol4KQ
- **Autor**: Matheus Battisti — canal "Hora de Codar"
- **Data publicação**: 2026-03-27
- **Duração**: ~17 min

## 📝 Resumo (suas palavras)
Battisti mostra como abrir o Claude Code num vault vazio do Obsidian e pedir, em um único prompt, a criação completa do segundo cérebro: estrutura de pastas, templates, regras (CLAUDE.md), skills oficiais do Kepano, skills customizadas e dados de exemplo. Defende que markdown local + LLM bate qualquer plugin proprietário porque o agente lê e edita `.md` nativamente. Demonstra também o uso de JSON Canvas para mapear conteúdos visualmente.

## 💡 Trechos-chave
- "Cria meu segundo cérebro" — prompt único que materializa o vault.
- 5 skills customizadas que sustentam o loop diário: nota diária, pesquisa, inbox, roteiro, manutenção semanal.
- Skills do Kepano são pré-requisito — sem elas, Claude gera markdown que quebra.

## 🪞 Reflexão
A grande sacada não é o vault — é o CLAUDE.md + skills. O vault em si é simples; o que faz funcionar é o "manual" que o agente lê antes de operar.
