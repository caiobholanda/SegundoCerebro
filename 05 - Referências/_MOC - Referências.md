# 📚 MOC — Referências

Material consultivo: livros, artigos, vídeos, docs, posts. **Não é cópia** — é resumo seu + links pra fonte original.

## Convenção
`Referência - Tipo - Título.md`

Exemplos:
- `Referência - Livro - Building a Second Brain.md`
- `Referência - Vídeo - Setup Obsidian Matheus Battisti.md`
- `Referência - Artigo - PARA Method by Tiago Forte.md`

```yaml
fonte: livro | artigo | video | podcast | doc | post
autor: Nome
url: https://...
tags: [referencia, tema-principal]
```
