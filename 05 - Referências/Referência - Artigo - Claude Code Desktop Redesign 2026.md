# Referência - Artigo - Claude Code Desktop Redesign 2026

## 📝 Resumo (suas palavras)
Anthropic redesenhou o app desktop do Claude Code para suportar múltiplas sessões paralelas em uma única janela. Foco em paralelismo: sidebar para gerenciar sessões, layout drag-and-drop, terminal integrado, editor de arquivos embutido e preview expandido.

## 💡 Trechos-chave
> Built to help you run more Claude Code tasks at once — new sidebar for managing multiple sessions, drag-and-drop layout for arranging your workspace, integrated terminal and file editor.

### Novos recursos do redesign
- **Sidebar** — lista todas as sessões ativas e recentes, filtrável por status/projeto/ambiente
- **Layout drag-and-drop** — arranja workspace conforme preferência
- **Terminal integrado** — roda testes/builds ao lado da sessão
- **Editor de arquivos in-app** — edições diretas sem sair do app
- **Diff viewer** — reconstruído para performance em changesets grandes
- **Preview expandido** — abre HTML, PDFs, servidores locais in-app
- **Side chat** (⌘+; / Ctrl+;) — branch de conversa sem contaminar contexto principal
- **Auto-archive** — sessão some da sidebar quando PR é mergeado/fechado

### Disponibilidade
- Todos os usuários: Pro, Max, Team, Enterprise, API

## 🪞 Reflexão
O redesign torna o Claude Code desktop mais próximo de uma IDE do que de um terminal. A separação entre sessão principal e side chat (que não adiciona contexto de volta) é especialmente útil para evitar desviar tarefas longas com perguntas pontuais.
