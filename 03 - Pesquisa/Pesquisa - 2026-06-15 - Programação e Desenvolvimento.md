# Programação e Desenvolvimento — 2026-06-15

## Rankings das Linguagens

| Ranking | Plataforma | Linguagem | Detalhe |
|---|---|---|---|
| #1 | GitHub (contribuidores) | TypeScript | 2,6M contribuidores/mês — 1ª vez na história (ago/2025) |
| #1 | TIOBE Index | Python | ~23,64% — dominância em AI/ML |
| Mais admirada | Stack Overflow | Rust | 9º ano consecutivo (72% aprovação) |

## TypeScript

- Ultrapassou Python no GitHub por contribuidores — mudança geracional consolidada
- Demanda: +66% YoY nos repos ativos
- Edge computing nativo: Cloudflare Workers, Vercel Edge Functions e Deno Deploy rodam TS nativamente
- Dev escolhe TypeScript porque "play well with AI coding assistants" e "catch errors before code runs"
- **94% dos erros de IA em código são tipo-relacionados** → tipagem estática é blindagem contra bugs de IA

## Python

- Posição #1 exclusivamente por domínio em IA/ML: PyTorch, TensorFlow, Hugging Face, LangChain
- Consistentemente maior demanda em job market junto com JavaScript/TypeScript

## Rust

- Reescrevendo sistemas críticos: OS, browsers, bancos de dados, cloud infra
- +35% vagas abertas em 2026
- 50% das empresas Fortune 500 usando Rust em produção
- Linux: 3/4 dos desenvolvedores Rust trabalham em infra server/cloud
- WASM + Rust crescendo para hot paths de performance

## AI-Native Development (Padrão 2026)

- **84%+** dos devs usam ferramentas de IA no workflow
- Papel do dev: **70% revisor/arquiteto** — criação manual de código deixou de ser atividade central
- Tiny Teams: indivíduo amplificado por IA consegue output antes de equipes inteiras
- DevSecOps + Green Software: de diferenciais → pré-requisitos
