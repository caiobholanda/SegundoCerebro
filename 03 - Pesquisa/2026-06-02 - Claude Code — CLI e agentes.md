# Pesquisa - Claude Code — CLI e agentes

## ❓ Pergunta central
Como o Claude Code utiliza a CLI e agentes para automação e integração em fluxos de trabalho?

## 🎯 Síntese (3-5 linhas)
O Claude Code combina uma interface de linha de comando (CLI) com agentes especializados para permitir uma automação flexível e eficiente de tarefas. A CLI oferece comandos intuitivos para interagir com o modelo, enquanto os agentes podem ser configurados para executar fluxos de trabalho específicos. Essa arquitetura é projetada para integração com diversas ferramentas e APIs externas, otimizando a produtividade de desenvolvedores e equipes.

## 🔬 Detalhes
- O Claude Code utiliza uma CLI que permite interagir diretamente com o modelo, fornecendo consultas, comandos e parâmetros de configuração.
- Os agentes são unidades especializadas que podem ser programadas para realizar tarefas específicas de forma autônoma, como mineração de dados, geração de relatórios ou integração com sistemas externos.
- A CLI suporta scripts parametrizados, permitindo que fluxos de trabalho repetitivos sejam automatizados e executados rapidamente.
- O sistema é compatível com APIs externas, facilitando integrações com ferramentas como GitHub, Jira, Slack, entre outras.
- Os agentes possuem um mecanismo de aprendizado constante, ajustando-se às mudanças nos fluxos de trabalho e otimizando a eficiência ao longo do tempo.
- A segurança é uma prioridade, com autenticação robusta e controle de permissões para evitar acessos não autorizados.
