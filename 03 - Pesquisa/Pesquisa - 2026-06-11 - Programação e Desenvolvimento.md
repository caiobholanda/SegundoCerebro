# Programação e Desenvolvimento — 2026-06-11

## Rankings de Linguagens

| Linguagem | Posição | Destaque |
|-----------|---------|---------|
| Python | #1 TIOBE (~23,64%) | Domina IA/ML/dados |
| TypeScript | #1 GitHub (contribuidores) | Desde agosto 2025 — 1ª vez na história |
| Rust | Top 10 | 9º ano mais admirada (Stack Overflow), +35% vagas |
| Go | Crescendo | Cloud-native, microserviços |

## TypeScript — Ascensão Histórica

- Ultrapassou Python no GitHub em ago/2025 por ~42.000 contribuidores
- **Dois motores**: frameworks modernos (Next.js, Angular) criam projetos TS por padrão + IA generativa acelerou adoção
- 80%+ dos novos projetos JS usam TypeScript
- Typagem estática cresce: 94% dos erros de IA são tipo-relacionados → TS mitiga

## Python — Domínio em IA/ML

- Continua #1 em IA, Machine Learning, Ciência de Dados e automação
- Sintaxe simples + ecossistema imbatível
- TIOBE: leve queda do pico 26,98% → ~23,64%, mas ainda dominante

## Papel do Desenvolvedor em 2026

- **70% do tempo**: revisão de código e arquitetura (não escrita)
- **30% do tempo**: escrita direta, cada vez mais assistida por IA
- Ferramentas como GitHub Copilot, Amazon CodeWhisperer, Claude Code = padrão (não diferencial)
- **84%+** dos devs usam IA no workflow diário

## Tendências Estruturais

- **DevSecOps**: segurança integrada desde o início — pré-requisito, não opção
- **Green Software**: programação sustentável emergindo como exigência corporativa
- **AI-Native Development**: novo padrão de trabalho consolidado
- **WASM + Rust**: crescendo para hot paths de performance

## Impacto do Fable 5 / Mythos 5 para Devs

- 1M context window → repositórios completos no contexto
- 128K output → refatorações maiores em uma única resposta
- $10/$50 (2× Opus) → context engineering ainda mais crítico para controle de custo
