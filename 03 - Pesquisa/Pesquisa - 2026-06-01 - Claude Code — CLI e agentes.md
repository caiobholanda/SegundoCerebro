# Claude Code — CLI e Agentes — 2026-06-01

## Destaque principal

Claude Opus 4.8 como padrão do Claude Code + **Dynamic Workflows** para orquestração de agentes em escala.

## Novidades da semana (26/05–01/06)

### Dynamic Workflows
- Orquestra numerosos subagentes a partir de um único script
- Permite abordagem de problemas em grande escala (decomposição massiva de tarefas)
- Integrado ao Claude Code como feature de produção (não mais só Research Preview)
- Verificação automática dos resultados vs suite de testes

### Plugin de Segurança
- Novo plugin de orientação de segurança revisão proativa das modificações de código em tempo real
- Detecta vulnerabilidades antes do commit

### Fast Mode no Opus 4.8
- $10/$50 por MTok — competitivo e 3× mais barato que Fast Mode anterior
- Disponível direto no CLI

### CLI `ant` — consolidação
- Cliente de linha de comando para Claude API
- Integração nativa com Claude Code
- Versionamento de recursos de API em arquivos YAML
- Interação mais rápida com a API sem precisar do IDE

### Android CLI 1.0 (Google I/O)
- Suporte explícito a Claude Code, OpenAI Codex, Gemini e Antigravity
- Análise semântica de código Android nativamente
- Compose previews e UI tests sem abrir Android Studio

### Agent View (Research Preview — contínuo)
- Dashboard CLI unificado para gerenciar múltiplas sessões paralelas
- Ainda em Research Preview desde 11/05

## Comparativo: Gemini CLI vs Claude Code

Em 2026 surgiu comparativo relevante: ambos são CLIs agentivas, mas Claude Code tem ecossistema de skills/hooks mais maduro; Gemini CLI tem integração mais profunda com ferramentas Google (Drive, Docs, Calendar).
