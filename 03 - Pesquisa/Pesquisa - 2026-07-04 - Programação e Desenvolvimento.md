# Programação e Desenvolvimento — 2026-07-04

## Rankings de linguagens — julho 2026

| Linguagem | TIOBE | GitHub | Destaque |
|-----------|-------|--------|----------|
| Python | #1 (~19%) | Top 2 | IA/ML dominância |
| TypeScript | #1 contribuidores | #1 | 80%+ projetos JS usam TS |
| JavaScript | Top 3 | Top 1 repositórios | base do ecossistema front |
| Rust | #12 | crescendo | +35% vagas YoY; 9º ano mais amado |
| Go | Top 10 | estável | infra, microsserviços |

## TypeScript

- **#1 GitHub por contribuidores** — mantém desde ago/2025 (2,6M+ contrib/mês)
- 80%+ novos projetos JavaScript usam TypeScript
- TS 6.0 lançado em mar/2026 (último baseado em motor JS)
- **Project Corsa confirmado**: reescrita em Go para TS 7.0 — 10× mais rápido no compilador
- 94% dos erros em código gerado por IA são tipo-relacionados → type-safety como filtro de qualidade

## Python

- Abaixo de 19% no TIOBE (recuo após pico de 26,14%)
- Mantém liderança em IA/ML, data science e automação
- Queda relativa por crescimento de TS e Rust, não por perda absoluta de uso

## Rust

- 9º ano consecutivo como "mais admirada" pelo Stack Overflow
- 82,2% de admiração (devs que usam querem continuar)
- +35–42% de vagas/ano — maior crescimento relativo
- Adotado em sistemas críticos por Microsoft, Google, Amazon, Meta
- 50% das empresas Fortune 500 com produção em Rust

## Tendências de desenvolvimento

- **Dev = 70% revisor/arquiteto** — IA escreve, humano valida e direciona
- 84%+ dos devs usam IA no workflow diário
- DevSecOps + Green Software: pré-requisitos para vagas sênior em 2026
- **AI-Native development**: agentes autônomos geram, testam e fazem deploy de código
- Type-safety venceu definitivamente: TS > JS, Rust > C++ em novos projetos de sistemas

## Node.js

- v22.18.0: type stripping nativo por padrão (sem build step para TS básico)
- Reduz fricção de adoção TypeScript em projetos Node
