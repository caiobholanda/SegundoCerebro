# Programação e Desenvolvimento — 2026-06-13

## Linguagens em Destaque

### TypeScript
- **#1 no GitHub** (GitHub Octoverse 2025) — 1ª vez na história, desde agosto/2025
- 2,6 milhões de contribuidores mensais — superou Python
- Stack Overflow: 38,5% dos devs usam TypeScript
- Todos os frameworks modernos usam TS por padrão

### Python
- **#1 no TIOBE** (~23,64%) — posição quase inteiramente sustentada pelo ecossistema de IA/ML
- Linguagem de PyTorch, TensorFlow, Hugging Face, LangChain e praticamente todo framework de IA
- Python + TypeScript dominam completamente o desenvolvimento moderno

### Rust
- **Trajetória fortemente ascendente** — supply-demand imbalance dos mais amplos do mercado
- Para sistemas, infraestrutura crítica, WASM e hot paths de performance
- 9º ano como linguagem mais admirada no Stack Overflow (72% aprovação)
- +35% em vagas abertas
- Maior salary ceiling para quem investe 6–12 meses de aprendizado

## Tendências Macro

### Type Safety Vencendo
- TypeScript > JavaScript, Rust > C/C++, Kotlin > Java
- Indústria se moveu decisivamente para linguagens com segurança de tipos em compile time
- 94% dos erros gerados por IA são tipo-relacionados — reforça a tendência

### AI-Native Development
- **84%+ dos devs** usam IA no dia a dia
- Papel do desenvolvedor: **70% revisor/arquiteto**, 30% escritor de código
- IA amplifica o indivíduo → "Tiny Teams" com capacidade de grandes times
- GitHub Copilot = baseline, não diferencial competitivo

### Linguagens em Integração com IA
- Linguagens que integram bem com workflows de IA (Python, TypeScript, Rust) crescem
- Linguagens que não integram estão estagnando

## Padrões de Stack

- **Frontend**: TypeScript + React + Next.js (dominante)
- **Backend**: Python (FastAPI/Django) ou TypeScript (Node.js)
- **Sistemas/Performance**: Rust, Go
- **IA/ML**: Python exclusivamente
