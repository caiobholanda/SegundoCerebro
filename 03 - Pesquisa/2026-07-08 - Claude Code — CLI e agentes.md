# Pesquisa - Claude Code — CLI e agentes

## ❓ Pergunta central
Como o Claude Code utiliza CLI e agentes para interação e automação eficaz?

## 🎯 Síntese (3-5 linhas)
Claude Code combina interfaces de linha de comando (CLI) com agentes autônomos para facilitar tarefas complexas e programáticas. CLI fornece controle direto e personalizável, enquanto os agentes podem tomar decisões e executar ações com base em parâmetros definidos. A integração entre ambos é essencial para fluxos de trabalho automatizados.

## 🔬 Detalhes
- CLI permite interação direta com Claude via comandos simples, permitindo execução de tarefas sem necessidade de interfaces gráficas.
- Agentes são entidades programadas para executar tarefas específicas, podendo ser configurados para operar de forma autônoma ou assistida.
- Claude Code utiliza CLI para configurar, executar e monitorar agentes, conectando lógica humana a capacidades automatizadas.
- A combinação de CLI e agentes facilita a automação de processos repetitivos, como busca de informações ou análise de dados.
- A segurança no uso de CLI é otimizada por Anthropic, garantindo que comandos e agentes não ultrapassem permissões definidas.
- CLI pode ser integrado com sistemas externos via API, ampliando as possibilidades de uso em ambientes corporativos e técnicos.
