# Pesquisa — Programação e Dev (maio 2026)

## Ranking de linguagens (2026)

### Líderes consolidadas

| Linguagem | Posição | Domínio principal |
|---|---|---|
| TypeScript | #1 GitHub (2025→) | Web frontend/backend, tooling |
| Python | #1 IA/Dados | ML, AI, scripts, automação |
| JavaScript | Top 3 | Web, mobile (React Native) |
| Rust | Ascensão | Sistemas, blockchain, segurança |
| Go | Crescendo | Cloud, DevOps, microserviços |
| Java | Corporativo estável | Enterprise, Android |
| C# | Corporativo estável | .NET, games (Unity) |

**TypeScript ultrapassou Python** em número de repositórios no GitHub em agosto de 2025. Python ainda lidera em projetos de IA e ciência de dados, mas o ecossistema JS/TS somado tem maior representatividade em atividades gerais.

### Rust — por que está crescendo

- Garantias de segurança de memória em compile time (sem garbage collector)
- Performance comparável ao C++
- Adoção em: kernel Linux, Chrome, Android, blockchain
- Linguagem mais "admirada" no Stack Overflow Survey consecutivamente

### Go — nicho cloud/DevOps

Go domina ferramentas de infraestrutura: Docker, Kubernetes, Terraform, Prometheus — todos escritos em Go. Ideal para CLIs, serviços HTTP de alta performance e sistemas distribuídos.

## Frameworks mais demandados

### Frontend
- **React**: maior gap entre oferta/demanda — mais procurado que devs disponíveis
- **Next.js**: padrão para React em produção (SSR + RSC)
- **Vue 3 / Nuxt**: alternativa popular no mercado europeu/brasileiro

### Backend
- **Node.js**: **maior lacuna de demanda** do mercado — empregadores precisam de mais devs do que estão disponíveis
- **Spring Boot (Java)**: dominante no enterprise brasileiro
- **FastAPI (Python)**: crescendo como padrão para APIs de IA

### Mobile
- **React Native**: desenvolvimento cross-platform (iOS + Android) com JavaScript
- **Flutter**: crescendo como alternativa com Dart

## IA como ferramenta padrão de desenvolvimento

A grande mudança de 2026: IA não é especialidade, é **infraestrutura** de todo desenvolvedor.

**Ferramentas consolidadas:**
- GitHub Copilot — sugestão de código inline
- Amazon CodeWhisperer — alternativa AWS
- Claude Code — agentic coding (escopo de tarefa inteiro)
- Cursor / Windsurf — IDEs com IA nativa

**Impacto no trabalho:**
- Geração de boilerplate: automática
- Testes unitários: sugeridos automaticamente
- Code review: assistido por IA
- Debugging: IA explica e sugere correções

O desenvolvedor 2026 passa mais tempo em **design de sistema, arquitetura e review** do que em digitação.

## Low-code / No-code em expansão

Ferramentas low-code democratizam criação de MVPs e automações internas. Não substitui devs para sistemas complexos, mas:
- Acelera prototipação
- Permite que equipes de negócio automatizem fluxos sem TI
- Reduz backlog de demandas pequenas

Plataformas em alta: Bubble, Retool, n8n (open-source), Make (ex-Integromat).

## WebAssembly (WASM) continua crescendo

- Execução near-native no browser e server-side
- Suporte a linguagens compiladas (Rust, C, Go) no frontend
- Cloudflare Workers usa WASM para edge computing
