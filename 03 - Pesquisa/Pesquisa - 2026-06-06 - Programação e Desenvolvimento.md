# Programação e Desenvolvimento — 2026-06-06

## Destaque principal

**Três grandes forças em 2026**: AI-Native Development (>84% devs usam IA), DevSecOps como pré-requisito (não mais diferencial) e Green Software emergindo como nova obrigação em contratos corporativos.

## Rankings de linguagens (consolidado junho 2026)

| Linguagem | Ranking | Tendência | Caso de uso primário |
|---|---|---|---|
| **Python** | #1 TIOBE (~23,64%) | Estável no topo | IA/ML, data science, scripts |
| **TypeScript** | #1 GitHub contribuidores | Crescendo | Web full-stack, projetos grandes |
| **JavaScript** | #2 GitHub | Estável | Web (ainda dominante no browser) |
| **Rust** | Mais admirada (8º ano) | Crescendo +35% vagas | Sistemas, WASM, segurança crítica |
| **Go** | Top 10 | Crescendo | Microserviços, cloud-native, CLI tools |

## AI-Native Development (junho 2026)

- >84% das equipes de desenvolvimento usam IA no fluxo diário
- GitHub Copilot, Amazon CodeWhisperer e Claude Code tornaram-se **baseline** (não diferencial)
- Papel do dev transformado: 70% do tempo em revisão/arquitetura, 30% em escrita de código
- Anthropic confirma: 80% do novo código em produção escrito por Claude
- Conceito "Tiny Teams": IA amplifica capacidade individual — equipes menores entregam mais

## Tendências técnicas

**Rust reescrevendo infraestrutura crítica:**
- Sistemas operacionais (componentes Linux, Windows)
- Browsers (componentes Firefox, Chrome)
- Bancos de dados
- WASM + Rust: combinação crescente para performance web de alto desempenho

**Python + Rust:**
- PyO3/maturin para hot paths: código Python chamar módulos Rust seamlessly
- Permite combinar rapidez de desenvolvimento Python com performance de Rust

**TypeScript como padrão:**
- Todos os frameworks modernos usam TS por padrão
- >80% dos novos projetos JavaScript já nascem em TypeScript

## DevSecOps + Green Software

- **DevSecOps**: security integrado no pipeline de CI/CD — passou de "diferencial" para pré-requisito em seleção de candidatos
- **Green Software**: código eficiente para reduzir consumo energético — emergindo em requisitos de contratos enterprise em 2026
- **WebAR com 5G**: realidade aumentada no browser sem app — crescendo com proliferação de 5G
