# Programação e Desenvolvimento — 2026-06-03

## Destaque principal

**TypeScript #1 no GitHub por contribuidores** desde ago/2025: +1M novos contribuidores em 2025. Dois vetores explicam o salto: frameworks modernos (Next.js, Angular) usam TS por padrão, e a IA generativa acelerou ainda mais a adoção. Python segue #1 em IA/dados e #1 TIOBE (~21-26%).

## Panorama atual (jun/2026)

### Rankings

| Linguagem | GitHub (contribuidores) | TIOBE | Stack Overflow |
|-----------|------------------------|-------|----------------|
| TypeScript | #1 (desde ago/2025) | Top 5 | Alta |

| Rust | Top 10 | Crescendo | #1 "mais admirada" (6º ano) |

### Tendências 2026

- **AI-Native Development**: GitHub Copilot como padrão de mercado — dev = 70% revisor/arquiteto, 30% escritor
- **DevSecOps**: de diferencial → pré-requisito em 2026
- **Green Software**: código eficiente para reduzir consumo energético — emergindo como padrão ESG tech
- **Rust**: +35% vagas, 6º ano consecutivo como linguagem "mais admirada"; crescendo em WASM + segurança
- **Go**: consolidado para microserviços cloud-native
- **WASM + Rust**: crescimento em ambientes browser e edge

### Frameworks dominantes

- Frontend: React + Next.js (com TS por padrão)
- Backend Python: FastAPI + Django
- Backend Node: Express + NestJS (TS)

## O que mudou desde 2026-06-02

- Confirmação de +1M novos contribuidores TS em 2025 — maior crescimento de linguagem documentado no GitHub
- GitHub Copilot consolidado como ferramenta padrão — não é mais diferencial, é baseline
