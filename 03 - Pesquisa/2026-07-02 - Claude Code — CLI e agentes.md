# Pesquisa - Claude Code — CLI e agentes

## ❓ Pergunta central
Como o Claude Code pode ser utilizado via CLI e integrado com agentes para automação avançada?

## 🎯 Síntese (3-5 linhas)
O Claude Code oferece ferramentas robustas para integração com linhas de comando (CLI) e agentes, permitindo automação de fluxos de trabalho complexos. A funcionalidade é ampliada por bibliotecas como o Anthropic SDK e por APIs bem documentadas. Esses recursos possibilitam desde a execução de tarefas isoladas até a integração com sistemas maiores, como pipelines DevOps e assistentes pessoais inteligentes.

## 🔬 Detalhes
- O Claude Code suporta interação via CLI, permitindo que desenvolvedores executem comandos, scripts e modelos baseados em IA diretamente do terminal.
- A integração com agentes permite criar fluxos de trabalho dinâmicos, onde o Claude pode agir como um orquestrador ou assistente em sistemas mais amplos.
- O Anthropic SDK é uma ferramenta chave para integrar o Claude com outras plataformas, fornecendo uma interface unificada para chamadas de API.
- A arquitetura é projetada para escalabilidade, sendo possível implementar soluções desde protótipos locais até sistemas distribuídos baseados em nuvem.
- O suporte a JSON e YAML facilita a configuração e o compartilhamento de fluxos de trabalho parametrizados, tornando o Claude Code altamente reutilizável.
- Casos de uso comuns incluem automação de tarefas repetitivas, geração de código sob demanda e suporte a decisões em tempo real com base em grandes volumes de dados.
