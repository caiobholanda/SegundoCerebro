# Pesquisa - Claude Code — CLI e agentes

## ❓ Pergunta central
Como os recursos de CLI e agentes no Claude Code podem ser integrados para automatização de tarefas e desenvolvimento de sistemas inteligentes?

## 🎯 Síntese (3-5 linhas)
Claude Code oferece suporte a CLI (Command Line Interface) e agentes como ferramentas fundamentais para automação e interação em sistemas. A CLI facilita a execução de comandos programáticos e a integração com fluxos de trabalho. Os agentes são modelos avançados que realizam tarefas autônomas baseadas em prompts e parâmetros definidos, permitindo que desenvolvedores criem soluções inteligentes e adaptáveis.

## 🔬 Detalhes
- **CLI no Claude Code**: A interface de linha de comando permite executar tarefas programáticas diretamente em terminais, integrando-se a pipelines de automação.
- **Agentes Claude**: São modelos que operam baseados em prompts e instruções, capazes de realizar tarefas diversas, como análise de dados, geração de textos e execução de comandos.
- **Flexibilidade dos agentes**: Claude oferece suporte para personalização de prompts e parâmetros, permitindo ajustes conforme os casos de uso específicos.
- **Integração com SDKs**: O Anthropic SDK pode ser usado junto ao CLI para facilitar a conexão entre os agentes e aplicações externas.
- **Automação e escalabilidade**: A combinação de CLI e agentes permite criar soluções que automatizam fluxos de trabalho e aumentam a eficiência em tarefas repetitivas.
- **Segurança e controle**: O uso de CLI aliado aos agentes respeita políticas de autorização e autenticação, garantindo que comandos e processos sejam executados de forma segura.
