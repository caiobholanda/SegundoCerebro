# Pesquisa - Claude Code — CLI e agentes

## ❓ Pergunta central
Como o Claude Code pode ser utilizado via CLI e integrado a agentes de automação?

## 🎯 Síntese (3-5 linhas)
O Claude Code, desenvolvido pela Anthropic, pode ser acessado via CLI para realizar tarefas como geração de texto, análise de dados e automação. Ele se conecta a agentes através de APIs, permitindo fluxos de trabalho personalizados. Além disso, sua interface de linha de comando suporta parâmetros configuráveis, o que facilita a integração com sistemas já existentes.

## 🔬 Detalhes
- O Claude Code pode ser acessado por meio de uma CLI, permitindo execução rápida de comandos diretamente no terminal.
- A integração com agentes é feita utilizando a API da Anthropic, que suporta tanto interações síncronas quanto assíncronas.
- É possível configurar prompts personalizados diretamente na CLI para ajustar o comportamento do modelo a diferentes contextos.
- A CLI oferece suporte a pipelines de automação, permitindo que respostas do Claude sejam reutilizadas em outros sistemas.
- Possui suporte a tokens de autenticação seguros para proteger acessos e integrações com agentes externos.
- A documentação oficial inclui exemplos para automação de tarefas repetitivas, como processamento de dados ou geração de relatórios.
