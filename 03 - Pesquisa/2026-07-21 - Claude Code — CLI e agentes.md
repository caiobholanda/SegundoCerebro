# Pesquisa - Claude Code — CLI e agentes

## ❓ Pergunta central
Como o Claude Code pode ser integrado a fluxos de trabalho via CLI e agentes automatizados?

## 🎯 Síntese (3-5 linhas)
O Claude Code permite automação poderosa de tarefas e fluxos de trabalho ao ser integrado com ferramentas de linha de comando (CLI) e agentes. Ele utiliza a Claude API e o Anthropic SDK para criar interações dinâmicas e executar comandos baseados em linguagem natural. Isso viabiliza desde integrações com sistemas existentes até a criação de agentes personalizados para tarefas técnicas e operacionais.

## 🔬 Detalhes
- O Claude Code é um sistema de IA desenvolvido pela Anthropic, projetado para interagir de forma conversacional e oferecer suporte em tarefas complexas.
- A integração com CLI permite que o Claude Code execute comandos diretamente em sistemas operacionais, expandindo sua utilidade em fluxos DevOps e automações.
- Agentes baseados no Claude Code podem ser configurados para desempenhar funções específicas, como monitoramento, deploys e integração contínua.
- O Anthropic SDK fornece bibliotecas e ferramentas para facilitar o desenvolvimento de aplicações que utilizam a Claude API.
- Agentes podem ser configurados para consumir dados de APIs externas, processá-los e responder a consultas de forma contextualizada, unindo fluxos de dados em tempo real.
- A segurança e a ética são pilares fundamentais no design do Claude, com limites explícitos para evitar mau uso e priorizar segurança em implementações.
