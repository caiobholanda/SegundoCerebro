# Inteligência Artificial — Avanços — 2026-06-05

## Destaque principal

**Microsoft Build 2026 (2–4/jun) redefiniu a estratégia de IA da empresa**: com MAI-Thinking-1 e MAI-Code-1-Flash, a Microsoft lança seus primeiros modelos proprietários treinados do zero, sinalizando movimento estratégico de independência em relação à OpenAI — ao mesmo tempo em que renegocia o acordo bilateral.

## Novidades (semana de 03–05/06)

### Microsoft Build 2026
- **MAI-Thinking-1**: primeiro modelo de raciocínio da Microsoft — 35B parâmetros ativos, 256k tokens de contexto, treinado com dados limpos e licenciados para uso comercial; foco em instruções multi-etapa complexas e geração de código
- **MAI-Code-1-Flash**: primeiro modelo da Microsoft que converte descrições textuais em código — testado com McKinsey, superou GPT-5.5 com 10× mais eficiência de custo
- **Copilot CLI**: Microsoft cancelou licenças corporativas do Claude Code (push interno para Copilot CLI como padrão empresarial)
- **Surface RTX Spark Dev Box**: 128GB RAM para IA local (anunciado no Build)

### OpenAI
- **GPT-Rosalind**: modelo atualizado para ciências da vida com agentes de codificação e descoberta de fármacos mais fortes — research preview confiável expandido globalmente
- **Renegociação com Microsoft**: OpenAI renegocia acordo e deve economizar US$97 bilhões em pagamentos — altera dinâmica do ecossistema

### Google DeepMind
- **Gemma 4 12B para laptops**: modelo local, multimodal, voltado para dispositivos pessoais — democratiza IA avançada sem nuvem
- **AGI timeline (Hassabis)**: DeepMind CEO mantém previsão de AGI até 2030 (±1 ano), definida como "humano-nível amplo em tarefas cognitivas"

### Tendência geral
- IA migra de **espetáculo para design de sistemas**: Samsung, Google DeepMind, OpenAI e Microsoft convergem para foco em implementação prática e operacionalidade em vez de demonstrações de capacidade bruta

## O que mudou desde 2026-06-03

- Microsoft Build 2026 concluído (2–4/jun) com anúncio de modelos próprios
- Renegociação OpenAI–Microsoft clarificada: $97B de economia projetada
- Microsoft cancelou licenças corporativas do Claude Code (favorece Copilot CLI)
