# Pesquisa - Claude Code — CLI e agentes

## ❓ Pergunta central
Como o Claude Code pode ser utilizado para criar e controlar agentes inteligentes via CLI?

## 🎯 Síntese (3-5 linhas)
Claude Code fornece uma interface robusta para integrar agentes de IA com ferramentas de linha de comando (CLI). Ele permite executar tarefas automatizadas, interagir com APIs e processar fluxos de trabalho complexos. Sua integração com o Anthropic SDK facilita a criação de agentes personalizados, enquanto sua flexibilidade o torna ideal para desenvolvedores e times focados em produtividade.

## 🔬 Detalhes
- Claude Code é uma extensão do Anthropic SDK, adaptada para uso em CLI e automação de fluxos de trabalho.
- Permite a criação de agentes que podem executar comandos no sistema operacional, interagir com outros aplicativos e até operar scripts.
- Suporta fluxos de entrada e saída de dados, permitindo que os agentes processem informações em tempo real.
- A configuração inicial requer autenticação via API keys fornecidas pela Anthropic.
- A linguagem natural pode ser usada para comandar os agentes, reduzindo a necessidade de sintaxes complexas.
- É possível criar agentes que monitoram processos, geram relatórios e interagem com sistemas remotos de maneira autônoma.
- Claude Code pode ser integrado a ferramentas populares de DevOps, como Docker, Kubernetes e Jenkins, para automação avançada.
- Suporta extensões para adaptar funcionalidades específicas às necessidades do usuário ou do projeto.
