# Programação e Dev — Novidades (2026-06-30)

## Ranking de Linguagens

### TypeScript — #1 GitHub (por contribuidores)
- Primeira vez na história supera Python no GitHub
- 2,6M+ contribuidores/mês (mantém desde ago/2025)
- 80%+ de novos projetos JavaScript já usam TypeScript
- Type-safety venceu definitivamente: TS > JS, Rust > C++, Kotlin > Java
- **94% dos erros em código gerado por IA são tipo-relacionados** → TypeScript reduz drasticamente erros de IA

### Python — #1 TIOBE (por popularidade geral)
- ~21,81% de market share (fev 2026), caiu levemente do pico ~23,64%
- Sinônimo de IA/ML: TensorFlow, PyTorch, scikit-learn dominam ecossistema
- #1 em vagas no Brasil para backend e dados
- FastAPI + Django como padrão de backend Python

### Rust — Crescimento Consistente
- 9º/10º ano consecutivo como linguagem "mais admirada" (Stack Overflow)
- +35% vagas YoY
- 50% das empresas Fortune 500 com Rust em produção (sistemas críticos)
- Microsoft, Google, Amazon, Meta adotando Rust para infraestrutura core

### Java + C# — Enterprise Sólido
- Preferidos para sistemas corporativos de larga escala
- Spring Boot (Java) e ASP.NET (C#) dominam enterprise backend
- Kotlin substituindo Java no Android

## Tendências Estruturais

### Dev = 70% Revisor/Arquiteto
A transformação do papel do desenvolvedor:
- 84%+ dos devs usam ferramentas de IA (GitHub Copilot, Copilot Chat, Claude Code)
- Foco mudou: geração de código → revisão, arquitetura, prompts de qualidade
- **Vibe coding** em ascensão: descrição em linguagem natural → código funcional
- Risco: acúmulo de dívida técnica se o dev não revisar criticamente

### DevSecOps — Pré-requisito
- Segurança integrada ao pipeline desde o início (não como etapa final)
- SAST/DAST automatizados em CI/CD
- Developers precisam entender vetores de ataque básicos

### Green Software — Carbono como Métrica
- Código eficiente = menos energia = menor pegada de carbono
- Frameworks de green coding emergindo
- Empresas grandes (Microsoft, Google) publicando Carbon Intensity de seus serviços

### AI-Native Development
- Ferramentas de IA como parte obrigatória do toolchain
- Claude Code, GitHub Copilot, Cursor, Windsurf como padrão
- Context window de 1M tokens (Fable 5) muda workflows: codebase inteira no contexto

## Para o Brasil Especificamente

- Python + TypeScript = combinação com maior empregabilidade
- Rust: escassez de profissionais cria prêmio salarial
- Cloud (AWS, GCP, Azure) + containerização (Docker/K8s) como baseline
- IA/ML skills valem +50% no salário médio
