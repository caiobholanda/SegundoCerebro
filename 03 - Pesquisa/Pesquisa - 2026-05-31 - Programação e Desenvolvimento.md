# Pesquisa - Programação e Desenvolvimento (2026-05-31)

## ❓ Pergunta central
Quais linguagens e frameworks lideram em 2026? Como a IA está mudando o papel do dev?

## 🎯 Síntese
Python mantém liderança no TIOBE (~21,81%) impulsionado por IA/ML. TypeScript #1 no GitHub desde agosto/2025 — maior deslocamento de linguagem em mais de uma década. Rust continua a mais admirada por 6 anos consecutivos (+35% vagas). IA é ferramenta padrão para >84% dos devs — papel do desenvolvedor migrou para revisor/arquiteto (~70% do tempo). React+Next.js dominam frontend; FastAPI+Django o backend Python.

## 🔬 Detalhes

### Rankings de Linguagens (2026)
| Linguagem | Destaque |
|-----------|----------|
| **Python** | #1 TIOBE (~21,81%); domina IA/ML/data |
| **TypeScript** | #1 GitHub contribuidores (1ª vez, ago/2025) |
| **JavaScript** | Base de toda web; frameworks migram para TS |
| **Rust** | #1 "mais admirada" 6º ano; +35% vagas; cresce em WASM |
| **Go** | Cresce cloud-native e microserviços |
| **Java/Kotlin** | Mobile (Android) e enterprise |

### Frameworks Mais Demandados
**Frontend:**
- React + Next.js (full-stack JS dominante)
- Vue 3 (alternativa mais leve)
- Svelte (crescendo em performance)

**Backend:**
- FastAPI (Python async, IA/ML)
- Django (Python web clássico)
- Spring Boot (Java enterprise)
- Express/Fastify (Node.js)
- Laravel (PHP)

### Tendências Principais
1. **AI-Native Development**: >84% dos devs usam IA no dia a dia; dev = revisor/arquiteto, não escritor
2. **DevSecOps**: segurança como pré-requisito desde o código — não opcional
3. **WebAssembly (WASM)**: Rust compila para WASM → performance nativa no browser
4. **Green Software**: eficiência energética como critério de código
5. **TypeScript em tudo**: todos os frameworks JS modernos usam TS por padrão

### Impacto da IA no Desenvolvimento
- Código gerado por IA: estimativa 40-70% do que vai para produção
- Dev como revisor/arquiteto: ~70% do tempo
- Engenharia de prompt: competência core para qualquer dev
