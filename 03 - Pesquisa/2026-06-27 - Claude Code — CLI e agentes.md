# Pesquisa - Claude Code — CLI e agentes

## ❓ Pergunta central
Como o Claude Code pode ser integrado a fluxos de trabalho via CLI e agentes personalizados?

## 🎯 Síntese (3-5 linhas)
O Claude Code, desenvolvido pela Anthropic, é uma interface de linha de comando (CLI) e um sistema de agentes projetado para facilitar a interação com modelos de linguagem de maneira programática e eficiente. Ele permite a automação de tarefas e integração com fluxos de trabalho complexos, sendo uma ferramenta poderosa para desenvolvedores e pesquisadores que buscam aumentar a produtividade e explorar o potencial dos modelos de IA.

## 🔬 Detalhes
- O Claude Code é uma extensão da API do modelo de linguagem da Anthropic, projetada para uso em CLI e automações.
- Ele suporta a criação de agentes personalizados, que podem ser treinados para executar tarefas específicas de maneira autônoma ou semi-autônoma.
- A CLI do Claude Code permite interações rápidas e eficientes com o modelo em ambientes de desenvolvimento, sem necessidade de interfaces gráficas.
- A ferramenta oferece suporte a fluxos de trabalho baseados em prompt chaining, facilitando o encadeamento de instruções e respostas.
- É possível integrar o Claude Code com sistemas e frameworks existentes, como CI/CD pipelines, para aumentar a eficiência dos processos.
- A segurança é uma prioridade, com controles robustos para evitar o uso indevido do sistema ou a geração de respostas inadequadas.
