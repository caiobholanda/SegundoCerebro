# Pesquisa - Claude Code — CLI e agentes

## ❓ Pergunta central
Como o Claude Code utiliza CLI e agentes para potencializar fluxos de trabalho e automação?

## 🎯 Síntese (3-5 linhas)
O Claude Code integra-se a fluxos de trabalho por meio de uma interface de linha de comando (CLI) robusta e agentes personalizáveis. Esses agentes são configurados para automatizar processos complexos, realizar tarefas repetitivas e interagir com APIs externas, tornando-se uma ponte entre usuários e sistemas. A combinação de CLI e agentes permite flexibilidade, escalabilidade e eficiência em projetos.

## 🔬 Detalhes
- O Claude Code CLI permite a criação, execução e gerenciamento de agentes diretamente pelo terminal, facilitando a integração em pipelines de desenvolvimento.
- Agentes no Claude Code são entidades programáveis que podem ser configuradas para executar tarefas específicas, como consultar APIs, processar dados ou interagir com usuários.
- A CLI suporta comandos para configurar, treinar e depurar agentes, além de oferecer suporte para execução em tempo real ou agendada.
- Os agentes podem ser programados para operar de forma autônoma, mas também permitem interação em tempo real com os usuários, ajustando seu comportamento com base no contexto das conversas.
- O sistema utiliza o Anthropic SDK para facilitar a integração com serviços externos e manipulação de dados, promovendo maior flexibilidade.
- A segurança é uma prioridade, com suporte para autenticação por chave API e controle de permissões para acesso a dados sensíveis.
