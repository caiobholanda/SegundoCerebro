# Pesquisa - Claude Code — CLI e agentes

## ❓ Pergunta central
Como o uso da CLI e de agentes expande as possibilidades do Claude Code em projetos de IA e automação?

## 🎯 Síntese (3-5 linhas)
O uso da CLI e de agentes no Claude Code permite integração mais profunda e customizada em fluxos de trabalho, facilitando automações e interações com modelos de IA. A CLI oferece comandos diretos para interagir com o modelo, enquanto os agentes permitem a orquestração de tarefas complexas. Juntos, eles ampliam as aplicações práticas e a acessibilidade para desenvolvedores e equipes técnicas.

## 🔬 Detalhes
- A CLI (Command Line Interface) do Claude Code suporta comandos para envio de prompts, ajustes de configurações e monitoramento de respostas diretamente do terminal.
- Agentes do Claude Code são scripts ou módulos que podem ser configurados para executar ações específicas, como buscar dados, realizar cálculos ou integrar-se com APIs externas.
- A CLI permite automação em pipelines de desenvolvimento, como testes de software e geração de documentação.
- Agentes podem ser usados para executar processos interativos com o modelo, como responder perguntas baseadas em documentos internos ou realizar análises de dados.
- A integração com ferramentas populares como Git, Jenkins e Docker é um dos diferenciais que torna a CLI do Claude Code poderosa para equipes de desenvolvimento.
- A combinação de CLI e agentes possibilita a criação de fluxos de trabalho totalmente automatizados, reduzindo erros manuais e aumentando a eficiência.
