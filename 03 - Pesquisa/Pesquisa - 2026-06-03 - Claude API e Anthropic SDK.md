# Claude API e Anthropic SDK — 2026-06-03

## Destaque principal

**Project Glasswing expande significativamente**: Anthropic estendeu acesso ao **Claude Mythos Preview** para ~150 novas organizações, adicionando também o **Claude Security** — varredura de base de código com sugestões de patch automatizadas. Lançamento público dos modelos Mythos-class previsto para "próximas semanas".

## Novidades (semana de 02–03/06)

- **Mythos Preview ativo em 150+ orgs**: Claude Security inclui varredura contínua de codebase + sugestões de patch; foco em zero-days em OS e browsers
- **Opus 4.8 consolidado**: modelo padrão em Max, Team Premium, Enterprise pay-as-you-go e API direta — sem alteração de preços ($5/$25 MTok)
- **Depreciação confirmada**: Claude Sonnet 4 e Claude Opus 4 serão removidos da API em **15/06/2026**
- **Fast Mode Opus 4.8**: 2,5× mais rápido, $10/$50 MTok (3× mais barato que versões anteriores)
- **Dynamic Workflows**: orquestração de centenas de subagentes paralelos em uma única sessão — verificação de saídas antes de retornar resultados
