# Programação e Desenvolvimento — 2026-06-09

## Rankings de Linguagens

| Linguagem | TIOBE | GitHub | Stack Overflow (mais amada) |
|---|---|---|---|
| Python | #1 (~21,81%) | Top 2 | Top 3 |
| TypeScript | Top 5 | **#1 (1ª vez)** | Top 5 |
| Rust | Top 10 | Crescendo | **#1 (8º ano consecutivo)** |
| JavaScript | Top 3 | Top 3 | — |

## Python

- Dominante em IA, ML, ciência de dados e automação
- Sintaxe simples + ecossistema de bibliotecas (NumPy, Pandas, PyTorch, FastAPI) impulsionam adoção
- Reinando absoluto fora do web development

## TypeScript

- Pela **primeira vez** como linguagem mais usada no GitHub — sinaliza mudança geracional no mercado
- 80%+ dos novos projetos JavaScript usam TypeScript
- "JavaScript com superpoderes" — segurança de tipos + escalabilidade em projetos grandes
- 94% dos erros de IA são tipo-relacionados → tipagem estática como resposta

## Rust

- **8º ano consecutivo** como linguagem mais amada (Stack Overflow Survey)
- Performance de C/C++ com segurança de memória garantida pelo compilador (sem garbage collector)
- Presente em produção em 50%+ das empresas Fortune 500
- +35% nas vagas de emprego em 2 anos
- Usado para reescrever infraestrutura crítica: SOs, browsers, bancos de dados

## Papel do Desenvolvedor em 2026

- **70% do tempo em revisão e arquitetura** — IA escreve o código, dev valida
- 84%+ das equipes de desenvolvimento usam IA ativamente
- GitHub Copilot passou de diferencial para baseline esperado
- Dev = revisor/arquiteto > escritor de código

## Tendências Consolidadas

- **DevSecOps** — segurança integrada ao pipeline desde o início (pré-requisito, não diferencial)
- **Green Software** — código eficiente para reduzir consumo de energia (emergindo como pré-requisito)
- **AI-Native Development** — IA como camada de infraestrutura padrão
- WASM + Rust crescendo para hot paths e workloads críticos
