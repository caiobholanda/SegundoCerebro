# Pesquisa - Claude Code — CLI e agentes

## ❓ Pergunta central
Como o Claude Code utiliza CLI e agentes para aprimorar fluxos de trabalho e interações com inteligência artificial?

## 🎯 Síntese (3-5 linhas)
O Claude Code integra inteligência artificial com ferramentas de linha de comando (CLI) para facilitar interações automatizadas e personalizáveis. Ele permite criar agentes que podem executar tarefas específicas, interagir com sistemas externos e processar comandos do usuário de maneira eficiente. Essa abordagem combina a flexibilidade do CLI com as capacidades avançadas de IA.

## 🔬 Detalhes
- O Claude Code utiliza bibliotecas como `Anthropic SDK` para interagir com APIs de IA e modelos como Claude.
- Agentes no Claude Code são projetados como entidades autônomas que podem tomar decisões baseadas em prompts e contexto fornecido.
- A integração com CLI permite que desenvolvedores configurem workflows personalizados, executando tarefas como parsing de texto, automação de rotinas e consultas dinâmicas.
- Scripts CLI podem ser configurados para enviar prompts diretamente ao modelo Claude, recebendo respostas formatadas para uso imediato.
- Um benefício significativo é a capacidade de conectar agentes a sistemas externos, como APIs REST, bancos de dados ou outras ferramentas de produtividade.
- O uso de CLI amplia o controle e a acessibilidade, permitindo que fluxos de trabalho de IA sejam usados tanto por desenvolvedores quanto por usuários técnicos com menos experiência.
