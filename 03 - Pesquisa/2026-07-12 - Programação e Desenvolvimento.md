# Pesquisa - Programação e Desenvolvimento

## ❓ Pergunta central
Quais são as tendências em linguagens e desenvolvimento de software em julho de 2026?

## 🎯 Síntese (3-5 linhas)
TypeScript consolida-se como linguagem padrão para scaffolding — vagas no LinkedIn BR cresceram 73% desde 2024. Python segue dominante em IA/ML mas perdeu espaço nos rankings gerais (recuou do pico de 26,98% no TIOBE para ~21%). Rust avança 35% em vagas/ano, sendo adotado em infra, cloud e blockchain. IA integrada ao dev (Copilot como padrão, não diferencial) reduz ciclos em 40% e transforma o dev em 70% revisor/arquiteto.

## 🔬 Detalhes
- **TypeScript**: #1 no GitHub (desde ago/2025, +1M contribuidores); vagas exigindo TS +73% no LinkedIn BR (2024-2026); sistemas de tipo forte capturam 94% dos erros gerados por IA antes de produção; TS 7.0 (Project Corsa) compilador em Go — 10× mais rápido (VS Code: 77,8s → 7,5s)
- **Python**: dominante em IA/ML, ciência de dados e automação; TIOBE recuou do pico de 26,98% para ~21%; ainda #1 ou #2 nos maiores rankings; imprescindível para engenharia de dados e LLMs
- **Rust**: vagas +35% a.a.; #12 TIOBE; 82%+ de "mais admirada" no Stack Overflow; cresce em infra, sistemas embarcados, cloud e blockchain; Go e Rust dividem espaço onde segurança e performance são inegociáveis
- **IA no desenvolvimento**:
  - GitHub Copilot virou padrão (não diferencial)
  - IA corta ciclo de desenvolvimento em ~40%
  - Dev vira 70% revisor/arquiteto, 30% escritor de código
  - 94% dos erros gerados por IA são tipo-relacionados → TypeScript essencial
- **Tendências gerais 2026**: DevSecOps (segurança integrada ao pipeline), programação sustentável (eficiência energética), WASM+Rust crescendo em produção
- **Meta-frameworks**: Next.js, Nuxt como entry points padrão; server-first com RSC/SSR dominante no React 19; Rolldown bundler (Rust-based) reduz builds de 2,5min para 40s
