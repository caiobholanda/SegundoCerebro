# Pesquisa - Claude Code — CLI e agentes

## ❓ Pergunta central
Como o Claude Code utiliza CLI e agentes para otimizar fluxos de trabalho e interação com IA?

## 🎯 Síntese (3-5 linhas)
O Claude Code expande a funcionalidade do Claude por meio de uma CLI que permite integração com sistemas locais e nuvem, além de automação de tarefas repetitivas. Agentes complementam essa abordagem ao executar tarefas específicas com base em instruções contextuais. Essa abordagem oferece flexibilidade e eficiência, facilitando a adoção de IA em fluxos de trabalho técnicos e não técnicos.

## 🔬 Detalhes
- O Claude Code inclui uma CLI que permite interação direta com o modelo através de comandos simples, integrando-se a ferramentas locais e servidores.
- A CLI suporta funcionalidades como chamadas de API, automação de tarefas, execução de scripts e manipulação de arquivos, aumentando a produtividade.
- Agentes podem ser configurados para executar tarefas específicas, como parsing de dados, geração de texto ou integração com bancos de dados.
- A modularidade é um dos pilares do sistema, permitindo adicionar funcionalidades por meio de plugins e scripts personalizados.
- O uso de agentes reduz a necessidade de supervisão humana constante, permitindo que o Claude realize tarefas de forma autônoma.
- A integração com sistemas existentes (como Git, AWS, Docker) torna a adoção do Claude Code acessível para desenvolvedores e equipes técnicas.
