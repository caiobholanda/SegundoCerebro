# Pesquisa - Claude Code — CLI e agentes

## ❓ Pergunta central
Como o Claude Code pode ser integrado via CLI e agentes para automação e fluxos de trabalho personalizados?

## 🎯 Síntese (3-5 linhas)
O Claude Code é uma ferramenta poderosa para automação e personalização de fluxos de trabalho por meio de integração com CLI e agentes. Ele permite a execução de comandos diretamente no terminal, facilitando o desenvolvimento e a interação com outros sistemas. Com suporte robusto a APIs e SDKs, possibilita a criação de agentes customizados para tarefas específicas, tornando-se uma solução versátil para desenvolvedores e equipes.

## 🔬 Detalhes
- O Claude Code pode ser integrado a linhas de comando (CLI) para executar tarefas programáticas diretamente do terminal.
- Ele suporta a criação de agentes customizados que interagem com APIs e outras ferramentas.
- O uso do Anthropic SDK é essencial para desenvolver integrações e agentes com funcionalidades mais avançadas.
- A estrutura modular do Claude Code facilita a adaptação a diferentes fluxos de trabalho e sistemas existentes.
- A segurança é um dos pilares da arquitetura do Claude Code, com suporte a tokens de autenticação e controle de permissões.
- Sua aplicação pode abranger desde automação de processos repetitivos até suporte em tempo real em sistemas baseados em chat.
