# Programação e Desenvolvimento — 2026-06-01

## Ranking de linguagens (junho 2026)

| Posição | Linguagem | Índice / Base |
|---------|-----------|--------------|
| #1 | Python | TIOBE ~21-26% (liderança em IA/dados/automação) |
| #1 | TypeScript | GitHub (contribuidores) — 1ª vez, desde ago/2025 |
| Top 3 | JavaScript | GitHub stars / projetos (TypeScript o está substituindo) |
| Crescimento | Rust | +35% vagas, 6º ano mais admirada |
| Cloud-native | Go | Microserviços, Kubernetes tooling |

**Marco histórico**: TypeScript superou Python no GitHub em ago/2025 (+66% YoY, ~1M novos contribuidores em 2025). Razão: frameworks modernos (Next.js, Angular, NestJS) usam TS por padrão.

## Estado atual do desenvolvimento

### AI-Native Development — padrão em 2026
- 84%+ dos desenvolvedores usam IA no trabalho diário
- GitHub Copilot, Amazon CodeWhisperer, Claude Code = stack padrão
- +80% dos novos projetos JavaScript usam TypeScript
- Dev = 70% revisor/arquiteto (menos escritor de código do zero)

### DevSecOps — de diferencial a pré-requisito
- Segurança integrada desde a concepção (não retroativa)
- "Shift left" consolidado: vulnerabilidades detectadas no PR, não em produção
- Ferramentas de SAST/DAST integradas ao CI/CD

### Green Software — emergindo como obrigatório
- Código eficiente = redução de consumo de energia = diferencial ESG
- Certificações de Green Software emergindo (Linux Foundation Green Software)
- Pressão crescente de grandes clientes e reguladores europeus

### WebAssembly + Rust — crescendo
- WASM permite Rust/C++ rodarem no browser com performance nativa
- Uso crescente em edge computing e aplicações de alta performance
- Rust: segurança de memória sem garbage collector = escolha para sistemas críticos

## Frameworks em destaque 2026

- **Frontend**: React + Next.js dominante; SvelteKit ganhando espaço
- **Backend Python**: FastAPI + Django (FastAPI crescendo por ser async-first e OpenAPI nativo)
- **Backend TypeScript**: NestJS, Hono, Elysia
- **Mobile**: React Native + Expo para indie; Flutter para enterprise Google
- **IA**: PyTorch + Hugging Face Transformers como stack padrão

## O papel do desenvolvedor em 2026

> Dev = revisor de código de IA + arquiteto de sistemas + engenheiro de prompt + avaliador de qualidade

Habilidades mais valorizadas:
1. Saber avaliar e corrigir código gerado por IA
2. Arquitetura de sistemas e decisões de design
3. Engenharia de prompt e context engineering
4. DevSecOps integrado
5. Raciocínio sobre custo e sustentabilidade
