# Pesquisa - Claude Code — CLI e agentes

## ❓ Pergunta central
Como o Claude Code pode ser utilizado via CLI para integrar agentes e fluxos automatizados?

## 🎯 Síntese (3-5 linhas)
Claude Code, desenvolvido pela Anthropic, oferece ferramentas para integração de IA em fluxos automatizados via CLI. O uso de comandos simplificados permite interação com modelos de linguagem e execução de tarefas. A integração em pipelines e agentes customizados é facilitada pela API Claude e pelo Anthropic SDK, ampliando possibilidades em sistemas dinâmicos.

## 🔬 Detalhes
- Claude Code utiliza comandos CLI para interagir diretamente com modelos de linguagem como Claude 2.
- O Anthropic SDK é uma peça-chave para quem deseja expandir as funcionalidades do Claude Code em sistemas mais complexos.
- O CLI suporta configuração de prompts personalizados, permitindo controle granular sobre as respostas da IA.
- Possui integração direta com sistemas de automação e agentes inteligentes, facilitando fluxos repetitivos ou dinâmicos.
- A segurança e controle ético são prioridades, com opções para limitar o escopo das interações via configuração.
- Claude Code pode ser usado em pipelines CI/CD para validação de código, geração de documentação e refatoração de scripts.
