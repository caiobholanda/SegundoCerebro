# Programação e Desenvolvimento — 2026-06-08

## Rankings de Linguagens

| Linguagem | Ranking | Destaque |
|---|---|---|
| Python | #1 TIOBE (~21,81%) | Queda do pico histórico 26,98%; domina IA/dados |
| TypeScript | #1 GitHub | +80% novos projetos JS usam TS; mais contribuidores |
| JavaScript | #2 GitHub | Base; substituído por TS em projetos novos |
| Rust | Top admired | Mais admirada pelo 8º ano; produção em 50% das empresas (+10pp em 2 anos) |
| Go | Crescendo | Cloud-native, microserviços, DevOps tooling |

## Tendências Estruturais

- **94% dos erros de código gerado por IA são tipo-relacionados** → tipagem estática (TypeScript, Rust, mypy) cresce como necessidade, não preferência
- **Dev = 70% revisor/arquiteto** — IA gera, humano valida; papel do desenvolvedor mais estratégico
- **AI-Native Development**: 84%+ dos devs usam IA no fluxo de trabalho; Copilot como baseline mínimo
- **Tiny Teams** — IA amplifica indivíduo; times de 2-5 pessoas entregam o que antes exigia 20+

## Rust em 2026

- Produção em 50% das empresas Fortune 500 (+10pp em 2 anos)
- Reescrevendo infraestrutura crítica: sistemas operacionais, browsers, bancos de dados
- WASM + Rust crescendo para web performance crítica
- PyO3/maturin: integração Python + Rust para hot paths

## Escolha por Objetivo

| Objetivo | Stack recomendada |
|---|---|
| Entrar rápido no mercado | TypeScript + SQL |
| IA e dados | Python + SQL |
| Cloud / DevOps | Go + Linux + automação |
| Sistemas / segurança | Rust |
| Mobile | Swift (iOS) / Kotlin (Android) |

## Práticas Emergentes como Padrão

- **DevSecOps** — segurança integrada desde o início do ciclo; não mais opcional
- **Green Software** — código eficiente como requisito; consumo energético de IA em evidência
- **Context Engineering** > Prompt Engineering — gerenciar o que entra no contexto dos LLMs é competência core para devs em 2026
