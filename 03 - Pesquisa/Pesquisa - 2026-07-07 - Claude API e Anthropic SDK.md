# Claude API e Anthropic SDK — 2026-07-07

## Limites Claude Code +50% (temporário até 13/jul)

Anthropic elevou os limites semanais do Claude Code em 50% para todos os planos Pro, Max, Team e Enterprise — vigente de 13/mai até **13/jul às 18h PDT**. Plano Free excluído.

**Motivo**: deal de compute com SpaceX (Colossus 1) — mais de 300 MW, 220k+ GPUs NVIDIA agora ativos. Contexto competitivo: OpenAI lançou o Codex em maio, ~4,2× mais eficiente em tokens em workloads similares.

Após 13/jul os limites revertem ao baseline anterior; planejar redução de uso ou upgrade de plano.

## Fable 5 — Volta Global (1/jul)

Claude Fable 5 retornou globalmente em 1/julho após suspensão por controles de exportação dos EUA. Volta com classificadores reforçados para ciberseg/bio/quím/destilação. Preço: $10/$50 por MTok.

## Fast Mode Opus 4.7 — Descontinuado

O modo rápido (`speed: "fast"`) do Opus 4.7 será **removido em 24/julho de 2026**. Requisições com esse parâmetro retornarão erro após essa data.

## Claude Sonnet 5 — Modelo Padrão

Claude Sonnet 5 é o modelo padrão em todos os planos (Free/Pro/Max/Team/Enterprise). Preço promocional: $2/$10 por MTok até 31/agosto, depois $3/$15.

## Analytics e Controles Enterprise

- Richer admin analytics com visibilidade por modelo
- Model-level entitlements e spend alerts
- Org default models via console admin

## Claude em Microsoft Azure (GA)

Claude disponível em GA no Microsoft Foundry/Azure com autenticação IAM nativa, billing integrado, governance e suporte a data zone US.
