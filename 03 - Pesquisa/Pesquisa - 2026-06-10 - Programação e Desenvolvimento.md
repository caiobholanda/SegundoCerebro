# Programação e Desenvolvimento — 2026-06-10

## Rankings de Linguagens 2026

| Linguagem | Ranking | Destaque |
|---|---|---|
| Python | #1 TIOBE (23,64%) | IA/ML/automação dominante |
| TypeScript | #1 GitHub (contribuidores) | 1ª vez na história — superou Python e JS |
| Rust | #1 mais amada (72% admiração) | 9º ano consecutivo — Stack Overflow |
| JavaScript | — | Base; migrando para TypeScript |
| Go | Crescendo | Cloud-native, microserviços |

## TypeScript — Mudança Geracional

- **#1 no GitHub por contribuidores** — primeira vez na história
- Considerado a linguagem padrão para desenvolvimento web profissional
- Frontend, backend, APIs, serverless e microserviços
- 80%+ dos novos projetos JS usam TypeScript por padrão
- 94% dos erros de IA são tipo-relacionados → tipagem estática se torna crítica

## Python — Domínio em IA

- TIOBE 23,64% (declínio leve do pico 26,98%)
- Central em AI/ML: TensorFlow, PyTorch, Pandas, NumPy
- Pré-requisito para qualquer papel de IA/dados

## Rust — Infraestrutura Crítica

- Reescrita em andamento: SO, browsers, bancos de dados, cloud infrastructure
- +35% nas vagas abertas nos últimos 2 anos
- Preferido para segurança e performance: memória safe sem GC
- Replacing C/C++ em infraestrutura crítica

## Papel do Desenvolvedor em 2026

> **Dev = 70% revisor/arquiteto** — IA gera código; humano valida, arquiteta, decide

- 84%+ dos devs usam IA no fluxo de trabalho
- Competências emergentes: context engineering, code review, AI orchestration
- DevSecOps + Green Software de diferencial → pré-requisito obrigatório

## Drivers Macro de Adoção

1. **IA como driver dominante**: linguagens que integram bem com AI workflows crescem
2. **Type safety ganha**: mercado migrando de JS → TS, C/C++ → Rust
3. **Performance crítica**: workloads de IA exigem código eficiente
