# Programação e Desenvolvimento — 2026-06-17

## Ranking de Linguagens 2026

| Linguagem | Ranking | Destaque |
|-----------|---------|----------|
| Python | #1 TIOBE (~23,64%) | IA/ML, dados, backend |
| TypeScript | #1 GitHub (contribuidores) | Web, frontend, backend seguro |
| JavaScript | Top 3 GitHub | Ainda dominante no mercado |
| Rust | 8º ano mais amado (Stack Overflow) | Sistemas, WASM, segurança |
| Go | Cresce em cloud-native | Microserviços, K8s |

## Python

- Domina IA/ML/dados — linguagem obrigatória para engenheiros de IA
- #1 TIOBE pelo segundo ano consecutivo em 2026
- FastAPI e Django como stacks backend mais adotadas

## TypeScript

- #1 GitHub por contribuidores (marco histórico, desde ago/2025)
- 80%+ dos novos projetos JS usam TS como padrão
- Migração JS→TS acelerada pela IA (94% dos erros de IA são tipo-relacionados)
- Todos os principais frameworks web adotam TS por padrão

## Rust

- 8º ano consecutivo como linguagem mais amada no Stack Overflow Survey 2026
- +35% de vagas abertas em relação ao ano anterior
- Em produção em 50%+ das empresas Fortune 500
- Casos de uso: sistemas operacionais, browsers, DBs, WebAssembly

## O Papel do Dev em 2026

- Dev passou a ser **70% revisor/arquiteto** e 30% escritor de código
- 84%+ dos desenvolvedores usam IA como ferramenta (GitHub Copilot como baseline)
- Demanda mudou: não basta escrever código, é preciso orientar agentes e revisar output
- Ferramentas como Cursor e Copilot democratizam linguagens menos conhecidas (C++, Rust)

## Tendências Técnicas

### DevSecOps — Pré-requisito
- Segurança integrada ao pipeline de desenvolvimento (não etapa separada)
- Obrigatório em 2026 — deixou de ser diferencial

### Green Software / Sustainable Computing
- Código eficiente para reduzir consumo de energia = tendência crescente
- Pressão de ESG sobre empresas de tech
- Decisões arquiteturais agora consideram footprint de carbono

### WASM + Rust
- WebAssembly crescendo com Rust como linguagem principal
- Permite código de performance nativa no browser
