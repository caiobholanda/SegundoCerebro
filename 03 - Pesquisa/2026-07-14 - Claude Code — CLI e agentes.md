# Pesquisa - Claude Code — CLI e agentes

## ❓ Pergunta central
Como o Claude Code pode ser utilizado via CLI e integrado a agentes para automação e personalização?

## 🎯 Síntese (3-5 linhas)
O Claude Code é uma ferramenta poderosa para interações via linha de comando (CLI), permitindo automatizar tarefas e integrar fluxos de trabalho em sistemas. Ele funciona como intermediário entre usuários e APIs, possibilitando a personalização de agentes que executam comandos específicos, respondem a consultas ou manipulam dados de forma programada. Sua integração com outros sistemas expande as possibilidades para uso em projetos complexos.

## 🔬 Detalhes
- O Claude Code pode ser acessado via CLI para interagir diretamente com APIs do Anthropic.
- Ele permite configurar agentes que executam tarefas específicas, como buscar dados, responder perguntas ou processar informações.
- A integração com scripts personalizados amplia a capacidade do Claude em fluxos de automação, como CI/CD ou monitoramento de sistemas.
- Utiliza comandos simples para o envio de prompts e recepção de respostas, sendo ideal para desenvolvedores e pesquisadores.
- Suporta entrada e saída de dados em diversos formatos, como JSON, para facilitar a integração com outros sistemas.
- A segurança é um ponto central, com autenticação de API e medidas de controle para evitar acessos não autorizados.
