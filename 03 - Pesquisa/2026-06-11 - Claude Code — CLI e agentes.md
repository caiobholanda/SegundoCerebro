# Pesquisa - Claude Code — CLI e agentes

## ❓ Pergunta central
Como o Claude Code pode ser utilizado via CLI para gerenciar agentes e integrar fluxos de trabalho automatizados?

## 🎯 Síntese (3-5 linhas)
O Claude Code oferece uma interface de linha de comando (CLI) que facilita a interação com agentes de IA, permitindo integração em pipelines automatizados e fluxos de trabalho diversos. Ele suporta comandos para criação, monitoramento e ajuste fino de agentes, além de extensões para customização de tarefas específicas. Sua flexibilidade o torna uma ferramenta poderosa para desenvolvedores e equipes técnicas.

## 🔬 Detalhes
- O Claude Code CLI é projetado para interagir diretamente com agentes Claude, seja para configuração inicial ou para comandos de execução em tempo real.
- É possível criar agentes personalizados via CLI, definindo parâmetros como prompts iniciais, comportamentos esperados e limites de operação.
- A CLI suporta integração com outras ferramentas DevOps, permitindo a inclusão de agentes em pipelines CI/CD ou automações em tempo de execução.
- Há suporte nativo para autenticação via chave API, garantindo segurança na execução de comandos e na comunicação com servidores Claude.
- Oferece comandos para monitorar logs, desempenho e métricas de uso de agentes, facilitando o ajuste fino e a análise de performance.
- Extensões permitem que desenvolvedores criem módulos customizados para tarefas específicas, ampliando o escopo de uso da CLI em diferentes domínios.
