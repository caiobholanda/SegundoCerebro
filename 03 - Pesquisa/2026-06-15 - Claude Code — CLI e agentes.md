# Pesquisa - Claude Code — CLI e agentes

## ❓ Pergunta central
Como o Claude Code pode ser utilizado via CLI para controlar agentes e integrar fluxos de trabalho?

## 🎯 Síntese (3-5 linhas)
O Claude Code permite a interação com agentes via CLI, oferecendo uma interface simplificada para automação de tarefas, integração com APIs e execução de comandos complexos. Ele suporta fluxos de trabalho personalizados, processamento de linguagem natural e execução de scripts em diversos ambientes. A flexibilidade do sistema o torna ideal para desenvolvedores e equipes que buscam otimizar suas operações.

## 🔬 Detalhes
- O Claude Code via CLI permite comandos interativos e execução de scripts diretamente no terminal.
- Suporta integração com APIs de terceiros, possibilitando extensões e automações personalizadas.
- Utiliza processamento de linguagem natural para interpretar e executar comandos complexos.
- Oferece suporte para múltiplos agentes, permitindo a coordenação de tarefas entre diferentes instâncias.
- Inclui mecanismos de segurança, como autenticação por chave API e permissões específicas para agentes.
- Pode ser integrado a CI/CD pipelines, aumentando a eficiência no desenvolvimento e na entrega de software.
