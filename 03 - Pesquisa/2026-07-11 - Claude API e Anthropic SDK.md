# Pesquisa - Claude API e Anthropic SDK

## ❓ Pergunta central
Quais são os modelos, preços e novidades da Claude API em julho de 2026?

## 🎯 Síntese (3-5 linhas)
Claude Sonnet 5 (lançado 30/jun) é o modelo padrão em julho de 2026, com preço promocional de $2/$10 por MTok até 31/ago, subindo para $3/$15 em setembro. O topo de linha é Claude Fable 5 ($10/$50 MTok) com Claude Mythos 5 ainda em acesso restrito. A Batch API oferece 50% de desconto e o prompt caching 90% off — tornando otimização de custo obrigatória em qualquer uso em escala.

## 🔬 Detalhes
- **Claude Sonnet 5**: modelo padrão Free/Pro/Max/Team/Enterprise; $2/$10 MTok até 31/ago → $3/$15 a partir de 1/set; contexto 1M tokens, 128k output
- **Claude Fable 5**: flagship público, $10/$50 MTok; contexto 1M, thinking adaptativo sempre ativo
- **Claude Mythos 5**: acesso restrito (Project Glasswing), mesmas specs que Fable 5
- **Claude Opus 4.8**: $5/$25 MTok
- **Claude Haiku 4.5**: $1/$5 MTok — melhor custo para tarefas simples
- **Batch API**: 50% de desconto em input e output para processamento assíncrono
- **Prompt caching**: 90% off em leituras do cache — maior alavanca de economia disponível
- **Contexto**: todos os modelos atuais suportam 1M tokens de contexto
