# Pesquisa - Claude Code — CLI e agentes

## ❓ Pergunta central
Como o Claude Code pode ser utilizado via CLI e quais são as melhores práticas para agentes que o integram?

## 🎯 Síntese (3-5 linhas)
O Claude Code, desenvolvido pela Anthropic, oferece integração via CLI para interagir com modelos de IA de maneira programática. Ele permite executar prompts, automatizar tarefas e criar agentes personalizados com comandos simples. Para desenvolver agentes eficazes, é essencial entender a estrutura de prompts, otimizar fluxos de trabalho e garantir a segurança no uso de APIs. 

## 🔬 Detalhes
- **Claude Code** é a interface de linha de comando (CLI) desenvolvida para facilitar o uso dos modelos da Anthropic, como o Claude 1 e 2.
- A CLI permite enviar prompts diretamente para o modelo e obter respostas em tempo real, sendo ideal para desenvolvedores que preferem trabalhar em terminais.
- **Agentes personalizados** podem ser criados utilizando o Claude Code em conjunto com ferramentas como scripts Python e APIs externas.
- O Claude Code suporta a integração com o **Anthropic SDK**, permitindo maior flexibilidade na criação de fluxos automatizados e agentes inteligentes.
- Boas práticas incluem a modularização dos fluxos de trabalho dos agentes, uso de logs para auditoria e segurança, e a implementação de limites para evitar chamadas excessivas à API.
- A documentação oficial da Anthropic é uma fonte essencial para entender as capacidades do Claude Code e suas configurações avançadas.
