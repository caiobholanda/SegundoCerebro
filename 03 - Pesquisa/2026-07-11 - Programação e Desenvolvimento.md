# Pesquisa - Programação e Desenvolvimento

## ❓ Pergunta central
Quais são as tendências em linguagens de programação e desenvolvimento em julho de 2026?

## 🎯 Síntese (3-5 linhas)
TypeScript mantém #1 no GitHub desde agosto de 2025, com mais de 1M novos contribuidores em 2025 (+66% YoY), e TypeScript 7.0 (compilador em Go, 10x mais rápido) redefine a experiência de desenvolvimento. Python caiu do pico de 26,98% TIOBE para ~21%, enquanto Rust cresce consistentemente (+35% vagas/ano) e foi novamente eleita a linguagem "mais admirada". A IA integrada ao workflow de dev é padrão — não diferencial.

## 🔬 Detalhes
- **TypeScript #1 GitHub**: desde ago/2025, superou Python por ~42k contribuidores; +1M novos contribuidores em 2025 (+66% YoY)
- **TypeScript 7.0 (Project Corsa/tsgo)**: compilador reescrito em Go; 10x mais rápido (VS Code: 77,8s → 7,5s para check completo); 3x menos memória; vagas TS BR +73% LinkedIn 2024-2026
- **Adoção TS**: frameworks modernos (Next.js, Angular) usam TS por padrão; 73% devs JS migraram; IA generativa acelerou adoção
- **Python**: ainda domina IA/ML e ciência de dados; mas caiu de pico 26,98% (jul/2025) para ~21% TIOBE — linguagens especializadas ganham nichos que eram do Python
- **Rust**: #12 TIOBE; "mais admirada" novamente (82,2% admiration); vagas +35% a.a.; ganhando espaço em sistemas críticos, cloud-native, segurança, embedded e WASM
- **94% dos erros em código gerado por IA são tipo-relacionados**: TypeScript com strict mode é camada de segurança essencial em projetos AI-assisted
- **Node.js v22.18**: type stripping nativo — TypeScript pode rodar diretamente sem transpilação em muitos casos
- **IA integrada**: IA é infraestrutura padrão, não diferencial; corta ciclo de dev ~40%
