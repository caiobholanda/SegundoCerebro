# Programação e Desenvolvimento — 2026-07-07

## TypeScript 7.0 — Project Corsa (RC disponível)

**RC lançado em 18/jun/2026.** Compilador completamente reescrito em Go (`tsgo`).

### Performance

| Projeto | TS 6.x | TS 7.0 (tsgo) |
|---------|--------|--------------|
| VS Code | 77,8s | **7,5s** |
| Playwright | 11,1s | **1,1s** |
| Memória | baseline | **3× menos** |

**Builds até 10× mais rápidos** com processamento paralelo nativo. Stable target: final de junho/início de julho 2026.

### Compatibilidade
TS 7 mantém compatibilidade com TS 6.0 em todos os casos exceto 74 (incompletos ou mudanças intencionais).

### Vagas
Vagas exigindo TypeScript no Brasil cresceram **73% no LinkedIn** entre 2024 e 2026.

## Python — Dominância em IA/ML

Python permanece #1 no TIOBE (abaixo do pico de 26,14%, mas consolidado). Linguagem padrão para IA/ML, data science e automação. Vagas com IA/LLMs+Python pagam 50% acima da média de mercado.

## Rust — Crescimento Contínuo

- **+35% vagas** ano a ano em 2026
- **Rust 1.95**: melhorias em compilação condicional e padrões de código (abr/2026)
- Microsoft, AWS, Google, Meta, Cloudflare reescrevendo componentes críticos em Rust
- 9º ano consecutivo como linguagem "mais admirada" no Stack Overflow Survey

## Tendências Gerais

- **94% dos erros de código gerado por IA** são tipo-relacionados → type safety é requisito
- **Dev = 70% revisor/arquiteto** — IA assumiu a geração de código rotineiro
- **84%+ dos devs** usam IA no workflow diário
- Rust e Go dominam infraestrutura onde segurança de memória é crítica
