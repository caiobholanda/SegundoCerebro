# Claude API — Novidades Junho 2026

## Novos Modelos Lançados

- **Claude Fable 5** (9 jun): disponível na API, AWS Bedrock, Vertex AI e Microsoft Foundry. State-of-the-art em quase todos os benchmarks — engenharia de software, visão, pesquisa científica. É um modelo da classe Mythos tornado seguro para uso geral.
- **Claude Mythos 5** (9 jun): disponível apenas para clientes aprovados no Project Glasswing. Possui as capacidades de cibersegurança mais fortes de qualquer modelo existente.

## Descontinuações

- Claude Sonnet 4 e Opus 4 foram retirados em **15 de junho de 2026**.

## Preços

- Fable 5 e Mythos 5: **$10/M tokens entrada** | **$50/M tokens saída**
- Menos da metade do preço do Mythos Preview anterior.

## Outras Mudanças

- **2 jun**: Anthropic parou de cobrar recusas que produzem saída vazia.
- **11 jun**: Limite de 90 segundos por célula de execução de código; novo parâmetro `response_inclusion` para web-tool.
