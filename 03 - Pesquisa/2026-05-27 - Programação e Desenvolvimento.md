# Pesquisa - Programação e Desenvolvimento

## ❓ Pergunta central
Como práticas modernas de programação e desenvolvimento impactam a produtividade e a qualidade do software?

## 🎯 Síntese (3-5 linhas)
A programação e o desenvolvimento de software evoluíram significativamente com o avanço das ferramentas, metodologias ágeis e práticas como DevOps. Esses elementos aumentaram a produtividade das equipes e a qualidade do código. No entanto, desafios como gerência técnica, escolhas de tecnologia e escalabilidade continuam sendo questões críticas.

## 🔬 Detalhes
- Metodologias ágeis, como Scrum e Kanban, ajudam a organizar equipes e priorizar entregas incrementais.
- Tecnologias como containers (ex.: Docker) e orquestração (ex.: Kubernetes) revolucionaram a implantação de sistemas escaláveis.
- Linguagens de programação continuaram evoluindo, com destaque para o uso crescente de Rust, Python e JavaScript/TypeScript.
- Práticas como revisão de código e testes automatizados são essenciais para garantir a qualidade e reduzir bugs.
- Ferramentas de integração e entrega contínua (CI/CD) como Jenkins e GitHub Actions aceleram o ciclo de desenvolvimento.
- Inteligência artificial está sendo integrada ao processo de desenvolvimento, como ferramentas de auto-completar e geração de código (ex.: GitHub Copilot).
