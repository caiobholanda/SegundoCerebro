# Pesquisa - Claude Code — CLI e agentes

## ❓ Pergunta central
Quais são as capacidades e limitações do Claude Code no uso de CLI e agentes?

## 🎯 Síntese (3-5 linhas)
Claude Code é uma ferramenta desenvolvida pela Anthropic que permite integrar modelos de IA em fluxos de trabalho usando CLI e agentes. Ele é projetado para ser flexível e seguro, mas possui algumas limitações de personalização e dependência de APIs. Seu uso é ideal para automações em operações repetitivas e suporte em desenvolvimento, mas requer atenção à sua configuração e às permissões de acesso.

## 🔬 Detalhes
- Claude Code é uma extensão do modelo Claude desenvolvida para automação via linha de comando (CLI) e agentes programáveis.
- A integração com CLI permite executar comandos diretamente do sistema operacional, facilitando tarefas como manipulação de arquivos, automação de builds e execução de scripts.
- Os agentes podem ser programados para interagir com APIs externas, permitindo execução de tarefas mais complexas, como coletar dados em tempo real ou realizar integrações com sistemas de terceiros.
- Possui suporte nativo para ambientes baseados em UNIX, mas a compatibilidade com Windows requer ajustes específicos.
- A segurança é um ponto forte, com políticas de controle de acesso que limitam comandos potencialmente perigosos, reduzindo o risco de mau uso.
- Atualmente, a capacidade de personalização dos agentes é limitada a templates e comandos pré-configurados, o que pode ser um entrave para aplicações altamente especializadas.
