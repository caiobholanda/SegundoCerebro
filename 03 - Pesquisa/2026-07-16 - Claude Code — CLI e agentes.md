# Pesquisa - Claude Code — CLI e agentes

## ❓ Pergunta central
Como o Claude Code utiliza CLI e agentes para integração e automação de fluxos de trabalho?

## 🎯 Síntese (3-5 linhas)
Claude Code, desenvolvido pela Anthropic, utiliza interfaces de linha de comando (CLI) como ponto de entrada para interagir com agentes especializados. Esses agentes são projetados para executar tarefas específicas, como automação de fluxos de trabalho, integração com APIs e suporte a desenvolvedores. A CLI permite flexibilidade e personalização, enquanto os agentes garantem precisão e eficiência nas interações, aproveitando a linguagem natural.

## 🔬 Detalhes
- O Claude Code combina o poder de modelos de linguagem avançados com interfaces CLI, permitindo que desenvolvedores interajam com agentes de IA diretamente em seus terminais.
- Agentes no Claude Code são unidades especializadas que podem ser configuradas para tarefas como análise de dados, automação de processos e integração com outras ferramentas, como GitHub e AWS.
- A interface CLI oferece comandos flexíveis que permitem aos desenvolvedores configurar, treinar e gerenciar agentes sem a necessidade de interfaces gráficas complexas.
- O design modular dos agentes possibilita que eles sejam facilmente conectados a sistemas existentes, facilitando a integração em ambientes corporativos.
- A segurança é um foco central, com autenticação integrada e gerenciamento de permissões para proteger informações sensíveis durante as interações.
- O uso de linguagem natural como input na CLI reduz a curva de aprendizado, permitindo que equipes multifuncionais colaborem diretamente com a ferramenta.
