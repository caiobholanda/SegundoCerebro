# Pesquisa - Claude Code — CLI e agentes

## ❓ Pergunta central
Como o Claude Code utiliza CLI e agentes para otimizar fluxos de trabalho e integração com sistemas?

## 🎯 Síntese (3-5 linhas)
O Claude Code usa a CLI para oferecer uma interface eficiente e programável, permitindo interações rápidas com agentes que podem ser personalizados para fluxos de trabalho específicos. Ele habilita automação e integração com ferramentas externas, simplificando tarefas complexas. A abordagem modular e extensível facilita o uso em diferentes cenários, com foco em desenvolvedores e equipes técnicas.

## 🔬 Detalhes
- A CLI do Claude Code permite interação direta com agentes por meio de comandos de terminal, simplificando o acesso às funcionalidades do sistema.
- Agentes podem ser configurados para atender a casos específicos, como análise de dados, assistência em escrita ou execução de tarefas repetitivas.
- A modularidade do sistema permite integrar o Claude Code com APIs externas, como o [[Claude API e Anthropic SDK]], expandindo suas capacidades.
- Os agentes utilizam modelos de linguagem avançados para interpretar comandos e fornecer respostas ou executar ações de forma contextual.
- A CLI oferece suporte a scripts e pipelines, permitindo automação de processos complexos sem necessidade de intervenção manual.
- Há um foco em segurança e privacidade, com logs de execução e limites configuráveis para evitar abusos ou erros críticos.
