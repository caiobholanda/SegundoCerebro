# Pesquisa - Claude Code — CLI e agentes

## ❓ Pergunta central
Como o Claude Code utiliza CLI e agentes para ampliar a funcionalidade e automação no Segundo Cérebro?

## 🎯 Síntese (3-5 linhas)
O Claude Code integra CLI e agentes para oferecer automação avançada no gerenciamento de notas e tarefas. CLI permite interação eficiente via linha de comando, enquanto agentes expandem capacidades contextuais, como responder perguntas e executar ações. A combinação melhora a produtividade e facilita fluxos dinâmicos no vault.

## 🔬 Detalhes
- O CLI do Claude Code serve como interface para comandos rápidos, permitindo criar, buscar e organizar notas via terminal.
- Agentes são unidades de processamento que interpretam e executam comandos complexos baseados em linguagem natural.
- A integração com o Anthropic SDK habilita o Claude Code a acessar modelos avançados de IA, como o Claude 3.
- A automação com agentes inclui tarefas como análise de lacunas, sugestão de conexões e atualização de notas, reduzindo trabalho manual.
- CLI e agentes podem ser configurados para rodar scripts personalizados que interagem com o vault.
- O uso de CLI e agentes melhora a interoperabilidade entre o Segundo Cérebro e outras ferramentas externas, como GitHub e APIs de produtividade.
