# Pesquisa - Claude Code — CLI e agentes

## ❓ Pergunta central
Como o Claude Code utiliza CLI e agentes para interagir com o usuário e automatizar tarefas no Obsidian?

## 🎯 Síntese (3-5 linhas)
O Claude Code usa uma combinação de CLI (Command Line Interface) e agentes para potencializar a automação e a interação em ambientes como o Obsidian. A CLI permite executar comandos diretamente no terminal, enquanto os agentes atuam como extensões inteligentes que interpretam e executam tarefas baseadas em linguagem natural. Essa integração é eficiente para fluxos de trabalho personalizados e automação avançada.

## 🔬 Detalhes
- O Claude Code utiliza CLI para permitir o controle direto de funcionalidades, como criar, editar e organizar notas no Obsidian.
- Agentes atuam como intermediários entre o usuário e o sistema, interpretando comandos de linguagem natural e traduzindo-os em ações específicas.
- A integração com ferramentas como o `obsidian-skills` expande as capacidades da CLI e dos agentes, permitindo automação programável e personalizável.
- A modularidade do Claude Code facilita a adição de novas funcionalidades, como bots especializados para tarefas específicas.
- A CLI e os agentes suportam fluxos de trabalho assíncronos, garantindo que múltiplas tarefas possam ser gerenciadas simultaneamente.
- A segurança é uma preocupação central, com autenticação e permissões configuráveis para evitar acessos não autorizados.
