# Pesquisa - Claude Code — CLI e agentes

## ❓ Pergunta central
Como o Claude Code pode ser utilizado via CLI para gerenciar agentes e fluxos de trabalho automatizados?

## 🎯 Síntese (3-5 linhas)
O Claude Code CLI é uma interface de linha de comando desenvolvida para facilitar a interação com agentes baseados em IA. Ele oferece comandos para criação, execução e monitoramento de agentes, além de integração com APIs e fluxos de trabalho. É uma ferramenta poderosa para desenvolvedores e equipes que buscam automação eficiente sem depender exclusivamente de interfaces gráficas.

## 🔬 Detalhes
- O Claude Code CLI é projetado para ser uma interface leve e acessível, permitindo que desenvolvedores interajam diretamente com a infraestrutura Claude.
- Suporta a configuração e gerenciamento de agentes personalizados, incluindo definições de comportamento e parâmetros de contexto.
- Integra-se nativamente com o Anthropic SDK, facilitando a comunicação com modelos Claude via API.
- Possui comandos específicos para registrar, listar, atualizar e excluir agentes, além de monitorar logs e métricas de desempenho.
- Oferece suporte para execução de fluxos de trabalho automatizados, permitindo a integração com outras ferramentas via pipelines.
- Inclui funcionalidades de segurança como autenticação API e controle de permissões para garantir a integridade dos fluxos de trabalho.
- A documentação oficial recomenda o uso do CLI em ambientes de desenvolvimento ágil, onde a automação de tarefas repetitivas é essencial.
