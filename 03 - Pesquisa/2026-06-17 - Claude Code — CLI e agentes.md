# Pesquisa - Claude Code — CLI e agentes

## ❓ Pergunta central
Como o Claude Code pode ser utilizado via CLI para integrar agentes inteligentes a fluxos de trabalho?

## 🎯 Síntese (3-5 linhas)
O Claude Code permite que agentes baseados na IA da Anthropic sejam integrados diretamente em fluxos de trabalho via CLI, facilitando automações e interações programáticas. A CLI oferece comandos para gerar, testar e iterar agentes, além de gerenciar APIs. Essa abordagem reduz a dependência de interfaces visuais e acelera o desenvolvimento de soluções personalizadas.

## 🔬 Detalhes
- O **Claude Code CLI** é uma interface de linha de comando para facilitar a interação com modelos da Anthropic, incluindo o Claude.
- Ele suporta a criação e execução de agentes virtuais, que podem realizar tarefas automáticas ou interagir com usuários.
- Permite configuração e personalização de agentes diretamente via arquivos YAML, simplificando a integração em pipelines de desenvolvimento.
- Comandos básicos incluem geração de prompts, testes de desempenho, e execução de workflows automatizados.
- É possível integrar o CLI com sistemas externos via APIs REST, permitindo que agentes interajam com outras aplicações.
- A ferramenta é útil tanto para desenvolvedores individuais quanto para equipes que buscam otimizar seus processos com IA.
