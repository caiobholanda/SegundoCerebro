# Programação e Desenvolvimento — 2026-05-30

## Rankings de Linguagens

### TIOBE Index (mai/2026)
| Posição | Linguagem | Market Share |
|---------|-----------|-------------|
| 1 | Python | ~26% |
| 2 | JavaScript/TypeScript | — |
| 3 | Java | — |

Python permanece dominante em IA/ML (TensorFlow, PyTorch, scikit-learn).

### GitHub (ago/2025 → referência 2026)
- **TypeScript**: #1 por contato de contribuidores — **1ª vez na história**
- Superou Python em ~42 mil contribuidores
- Crescimento de +1M novos colaboradores em 2025 (+66% YoY)

## Linguagens em Destaque

- **TypeScript**: #1 GitHub, padrão de frontends e backends modernos; todos os frameworks adotam por padrão
- **Python**: #1 TIOBE, domina IA/ML, data science e scripting
- **Rust**: mais admira pelo 6º ano consecutivo; cresce em sistemas críticos, cloud-native, segurança e embarcados; +35% em vagas
- **Go**: preferido para microserviços e cloud-native
- **Java**: mantém relevância corporativa (JDK 21 LTS)

## Tendências de Desenvolvimento 2026

### AI-Native Development
- 84%+ dos devs usam assistentes de IA diariamente
- Papel do dev migra: escritor de código → revisor/arquiteto
- Ferramentas de IA integradas nas IDEs como padrão (Copilot, CodeWhisperer, Claude Code)
- Engenharia de prompt como competência core para devs

### Segurança
- **DevSecOps**: de diferencial → pré-requisito obrigatório
- Shift-left: segurança desde o início do ciclo de desenvolvimento
- OWASP top 10 e análise estática como parte do CI/CD padrão

### Sustentabilidade
- **Green Software**: desenvolvimento com baixo consumo energético; de tendência → pré-requisito em projetos enterprise

### WebAssembly
- **WASM + Rust**: crescimento expressivo para aplicações web de alta performance e plugins cross-platform
