# Pesquisa - Claude Code — CLI e agentes

## ❓ Pergunta central
Como o Claude Code pode ser utilizado via CLI para integrar agentes em fluxos de trabalho automatizados?

## 🎯 Síntese (3-5 linhas)
O Claude Code CLI é uma ferramenta poderosa para interagir com o modelo de linguagem da Anthropic diretamente do terminal, permitindo a criação e gestão de agentes personalizados. Ele viabiliza fluxos de trabalho automatizados e integrações com outros sistemas, oferecendo flexibilidade através de comandos simples e suporte para extensões customizadas. No entanto, explorar melhor as limitações e as maneiras de gerenciar múltiplos agentes ainda é essencial.

## 🔬 Detalhes
- O Claude Code CLI permite interagir com os modelos da Anthropic via linha de comando, facilitando o uso em scripts e automações.
- É possível criar agentes personalizados com configurações específicas, como tom, conhecimento pré-carregado e instruções de comportamento.
- Suporta integração com outras ferramentas e APIs, tornando-o útil para desenvolvedores que desejam incorporar IA em sistemas existentes.
- O CLI oferece suporte para a execução de tarefas complexas, como geração de texto, análise de dados e até mesmo execução de código em linguagens específicas.
- A segurança e a privacidade são priorizadas, com opções para limitar o acesso e uso dos agentes criados.
- Existe a possibilidade de estender as funcionalidades do CLI através de plugins ou scripts customizados, permitindo adaptações a casos de uso específicos.
