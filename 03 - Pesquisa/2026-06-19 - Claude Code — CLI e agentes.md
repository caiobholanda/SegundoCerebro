# Pesquisa - Claude Code — CLI e agentes

## ❓ Pergunta central
Como o Claude Code pode ser utilizado via CLI para integração com agentes e automação de tarefas?

## 🎯 Síntese (3-5 linhas)
O Claude Code oferece um CLI (Command Line Interface) robusto para interagir com a API da Anthropic, permitindo a criação e gerenciamento de agentes de IA. Ele simplifica fluxos de automação, oferece suporte a gerenciamento de prompts e facilita a integração em scripts e sistemas existentes. Sua flexibilidade é ideal para desenvolvedores que buscam uma abordagem programável e ágil para gerenciar agentes autônomos.

## 🔬 Detalhes
- O CLI do Claude Code permite enviar prompts diretamente para a API da Anthropic, retornando respostas em tempo real.
- Suporta a configuração de agentes autônomos via comandos simples, com opções para ajustar parâmetros como temperatura, limite de tokens e contexto.
- Integração fácil com sistemas operacionais baseados em Unix, permitindo automação de tarefas por meio de scripts shell.
- Oferece ferramentas para debugging de prompts, facilitando a construção iterativa de fluxos de interação com agentes.
- Suporte a múltiplos formatos de saída, como JSON e texto simples, para integração com diferentes ferramentas e workflows.
- Pode ser utilizado em conjunto com frameworks de agentes autônomos, como LangChain, para criar pipelines complexos de processamento.
