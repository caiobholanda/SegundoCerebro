# Pesquisa - Claude Code — CLI e agentes

## ❓ Pergunta central
Como o Claude Code pode ser utilizado por meio de CLI e agentes para automatizar fluxos de trabalho complexos?

## 🎯 Síntese (3-5 linhas)
O Claude Code, aliado a interfaces de linha de comando (CLI) e agentes, permite a automação de tarefas repetitivas e a integração de sistemas complexos. Ele se destaca pela sua capacidade de processar linguagem natural, permitindo interações mais humanas e ágeis. Além disso, o uso de agentes possibilita a execução de tarefas específicas de forma autônoma, ampliando as possibilidades de aplicação.

## 🔬 Detalhes
- O Claude Code oferece um modelo de linguagem natural avançado, que pode ser acessado programaticamente via API ou integrado em ferramentas de CLI.
- Os agentes do Claude são programados com prompts personalizados para executar tarefas autônomas, como análise de dados ou geração de relatórios.
- A CLI facilita a integração do Claude Code em pipelines de automação, permitindo que processos sejam acionados diretamente de terminais ou scripts.
- O uso de "prompt chaining" em agentes permite a execução de tarefas sequenciais, onde a saída de uma etapa serve como entrada para a próxima.
- O Claude Code é otimizado para segurança e privacidade, com configurações que limitam o acesso a dados sensíveis durante a execução de tarefas automatizadas.
- Ferramentas como o Anthropic SDK podem ser usadas para criar e gerenciar agentes Claude em sistemas locais ou na nuvem.
