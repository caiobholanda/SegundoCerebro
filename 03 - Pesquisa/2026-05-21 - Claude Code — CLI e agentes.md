# Pesquisa - Claude Code — CLI e agentes

## ❓ Pergunta central
Como o Claude Code utiliza CLI para interagir com agentes e quais são suas principais funcionalidades?

## 🎯 Síntese (3-5 linhas)
O Claude Code CLI é uma interface de linha de comando desenvolvida para facilitar a interação com agentes baseados no modelo Claude da Anthropic. Ele permite automação, execução de tarefas específicas e integração com sistemas existentes. As principais funcionalidades incluem envio de prompts, gestão de sessões e análise de respostas. Sua arquitetura foca na simplicidade de uso e na flexibilidade para desenvolvedores e equipes técnicas.

## 🔬 Detalhes
- O Claude Code CLI é uma ferramenta que permite aos desenvolvedores interagir diretamente com a API do Claude via terminal.
- Ele suporta scripts automatizados, tornando-se útil para testes, prototipagem e fluxos de trabalho repetitivos.
- As funcionalidades básicas incluem envio de prompts, configuração de parâmetros, como temperatura e contexto, e exportação de respostas em formatos personalizados (JSON, Markdown, etc.).
- O CLI pode ser configurado para autenticar diretamente com a API usando chaves privadas, garantindo segurança em integrações.
- Permite a criação e gerenciamento de agentes personalizados, ajustando comportamentos e limites de interação.
- Suporta logs detalhados de sessões, úteis para depuração e análise de desempenho dos agentes.
- Integra-se com ferramentas como cron jobs e pipelines de CI/CD para execução automática de tarefas.
