# Pesquisa - Claude Code — CLI e agentes

## ❓ Pergunta central
Como o Claude Code permite a integração de agentes via CLI, e quais são suas melhores práticas de uso?

## 🎯 Síntese (3-5 linhas)
O Claude Code é uma interface de linha de comando (CLI) projetada para interação com agentes baseados em IA da Anthropic. Ele permite a execução de tarefas automatizadas, integração com pipelines de desenvolvimento e interação com APIs externas. O foco principal é simplificar fluxos de trabalho complexos e expandir as capacidades de agentes de IA em ambientes técnicos.

## 🔬 Detalhes
- O Claude Code é parte do ecossistema Anthropic, que inclui o [[Claude API e Anthropic SDK]].
- Oferece uma CLI para configurar, treinar e executar agentes de IA diretamente a partir do terminal.
- Suporta integração com outros sistemas via APIs RESTful, facilitando a comunicação entre agentes e serviços externos.
- Permite a automação de tarefas repetitivas ou complexas, como análise de dados, geração de código e processamento de linguagem natural.
- Inclui suporte a plugins modulares, permitindo a personalização e extensão das funcionalidades dos agentes.
- Oferece documentação extensa e exemplos práticos para desenvolvedores, além de suporte para linguagens como Python e JavaScript.
