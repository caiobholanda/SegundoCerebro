# Pesquisa - Claude Code — CLI e agentes

## ❓ Pergunta central
Como o Claude Code, por meio de CLI e agentes, pode ser utilizado para automatizar fluxos de trabalho e interações com IA?

## 🎯 Síntese (3-5 linhas)
O Claude Code utiliza interfaces de linha de comando (CLI) e agentes para oferecer uma solução programável e modular de integração de IA. A CLI permite automações locais rápidas e personalizadas, enquanto os agentes expandem a funcionalidade para interações mais dinâmicas e baseadas em contexto. Juntos, eles habilitam fluxos de trabalho eficientes e personalização profunda de aplicações.

## 🔬 Detalhes
- O **Claude Code CLI** é uma interface de linha de comando que permite que desenvolvedores integrem e interajam diretamente com o modelo Claude em seus fluxos de trabalho.
- O **sistema de agentes** no Claude Code permite criar agentes que podem processar consultas complexas, gerenciar estados e interagir com múltiplas fontes de dados.
- A CLI suporta execução de tarefas repetitivas programadas, como análise de texto, geração de relatórios e extração de insights.
- Os agentes podem ser configurados para serem altamente especializados em domínios específicos, como suporte técnico ou consultas financeiras.
- É possível combinar o uso da CLI com scripts personalizados para integrar Claude a outros sistemas, como CRMs, gerenciadores de projetos e ferramentas de automação.
- A abordagem modular do Claude Code facilita sua adaptação para diferentes casos de uso, desde projetos individuais até aplicações empresariais.
