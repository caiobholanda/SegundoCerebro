# Pesquisa - Claude Code — CLI e agentes

## ❓ Pergunta central
Como o Claude Code pode ser utilizado via CLI para interações avançadas com agentes e automações?

## 🎯 Síntese (3-5 linhas)
O Claude Code, desenvolvido pela Anthropic, oferece uma interface de linha de comando (CLI) que permite interações programáticas com agentes de IA. Essa abordagem facilita integrações em pipelines de automação, prototipagem e até mesmo no uso cotidiano de ferramentas como o Obsidian. Ele se destaca pela flexibilidade e pela possibilidade de customização, permitindo que desenvolvedores criem fluxos altamente específicos.

## 🔬 Detalhes
- O Claude Code fornece uma CLI que suporta comandos para criar, configurar e interagir com agentes de IA.
- A comunicação com agentes é baseada em prompts estruturados, permitindo controle granular sobre entradas e saídas.
- Suporta integração com outros sistemas via APIs, possibilitando uso em pipelines de CI/CD e automação de tarefas.
- É possível criar agentes customizados que executam funções específicas, como análise de textos, escrita automatizada e geração de código.
- A CLI do Claude Code é projetada para ser extensível, permitindo que desenvolvedores adicionem funcionalidades específicas através de plugins.
- Um ponto forte do sistema é a atenção à segurança e à ética, com salvaguardas embutidas para evitar usos inadequados.
