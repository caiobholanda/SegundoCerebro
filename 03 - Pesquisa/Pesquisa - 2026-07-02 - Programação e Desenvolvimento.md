## TypeScript — A Guerra Acabou

- Debate TS vs JS **encerrado**: TypeScript venceu para uso profissional
- **73% dos devs JS** já migraram (2026)
- 80%+ de todos os projetos JS novos usam TypeScript
- Ferramentas type-safe end-to-end: tRPC + Zod + Drizzle ORM
- **TypeScript 6.0** lançado (mar/2026) — último baseado em JavaScript
- **Project Corsa** confirmado: TypeScript 7.0 reescrito em Go (**10× mais rápido**)
- 94% dos erros em código gerado por IA são tipo-relacionados → tipagem estática = escudo

## Python — Queda do Pico, mas Ainda Dominante em IA

- TIOBE caiu abaixo de **19%** (pico: 26,14%)
- Permanece #1 em IA/ML com 41,9% dos devs querendo usá-lo
- Posição TIOBE #1 por mais de 3 anos consecutivos
- Recuo indica diversificação do ecossistema, não abandono

## Rust — Crescimento Consistente

- **82,2% de taxa de admiração** (mais amada pelo 9º ano)
- **83%** dos usuários atuais querem continuar usando
- +42% de vagas por ano; 50% das empresas Fortune 500 em produção
- Domínio crescente em: sistemas operacionais, browsers, databases, cloud infra
- Substituindo C e C++ em infraestrutura crítica

## Node.js

- **v22.18.0**: type stripping ativo **por padrão** — executa TypeScript sem build step
- Converge TypeScript como linguagem de runtime nativa no ecossistema Node

## Macro Tendências

- **IA como driver de adoção de linguagens**: Python, TypeScript e Rust crescem pela integração com IA
- **Type safety venceu definitivamente**: TS>JS, Rust>C/C++, Kotlin>Java
- **Dev = 70% revisor/arquiteto**: IA gera o código, humano revisa e decide
- 84%+ dos devs usam IA no workflow diário
- **DevSecOps + Green Software** como pré-requisitos para vagas sênior

## Vibe Coding

- Movimento "vibe coding" em ascensão: dev descreve intenção, IA escreve
- Torna type-safety ainda mais crítica: erros de tipo são a principal falha do código IA
