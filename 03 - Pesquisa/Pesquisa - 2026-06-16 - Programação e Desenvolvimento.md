# Programação e Desenvolvimento — 2026-06-16

## Linguagens Dominantes

### Python
- #1 TIOBE (~23,64%) — domínio absoluto em IA/ML/data
- Lidera vagas abertas no Brasil junto com JavaScript/TypeScript
- Crescimento acelerado pelo boom de IA

### TypeScript
- #1 GitHub por contribuidores (desde ago/2025)
- **80%+ dos novos projetos JavaScript usam TypeScript** em 2026
- Equipes migram de JS puro para TS para reduzir bugs em produção
- 94% dos erros de IA são tipo-relacionados → tipagem estática vence

### Rust
- 9º ano consecutivo mais admirada (Stack Overflow)
- Adotado por Microsoft, Google, Amazon e Meta para sistemas críticos de performance
- +35% crescimento em vagas
- Presente em 50%+ das empresas Fortune 500

## O Desenvolvedor em 2026

- **70% do tempo em revisão e arquitetura** — não em escrever código do zero
- 84%+ dos devs usam IA como parte do workflow
- GitHub Copilot como linha de base (não mais diferencial)
- Dev = revisor/arquiteto/orquestrador de agentes

## Tendências Técnicas

- **DevSecOps**: segurança integrada ao ciclo de desenvolvimento — pré-requisito, não diferencial
- **Green Software**: código eficiente para reduzir consumo de energia — tendência crescente em 2026
- **WASM + Rust**: crescendo para aplicações de alta performance no browser
- Todos os frameworks modernos usam TypeScript por padrão

## IA no Desenvolvimento

- Desenvolvimento AI-native é o novo padrão
- Claude Code, GitHub Copilot, Gemini Code Assist como ferramentas padrão
- Fable 5 com 1M ctx muda workflows de coding (codebase inteiro no contexto)
