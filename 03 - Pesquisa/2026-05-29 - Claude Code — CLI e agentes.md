# Pesquisa - Claude Code — CLI e agentes

## ❓ Pergunta central
Como o Claude Code - CLI e agentes podem ser utilizados para otimizar fluxos de trabalho e automação?

## 🎯 Síntese (3-5 linhas)
O Claude Code — CLI e agentes são ferramentas que integram o modelo de linguagem Claude da Anthropic com fluxos de trabalho automatizados. O CLI permite interações diretas com o sistema via linha de comando, enquanto agentes podem automatizar tarefas complexas em resposta a comandos naturais. Ambas as ferramentas são projetadas para desenvolvedores e usuários técnicos, promovendo eficiência e integração com outras ferramentas.

## 🔬 Detalhes
- O Claude Code - CLI é uma interface de linha de comando que permite interagir diretamente com o modelo Claude, ideal para desenvolvedores que preferem um ambiente terminal.
- Agentes Claude são instâncias configuráveis do modelo que podem executar ações automatizadas baseadas em entradas em linguagem natural.
- Ambos os recursos são construídos para permitir a integração com APIs externas, sistemas internos e workflows personalizados.
- O CLI suporta comandos configuráveis e scripts para tarefas repetitivas, como análise de dados e automação de processos administrativos.
- Os agentes podem ser ajustados para executar tarefas com base em prioridades específicas, permitindo decisões contextuais em tempo real.
- Ambos os sistemas utilizam princípios de alinhamento ético e segurança, seguindo as diretrizes da Anthropic para evitar usos indevidos.
