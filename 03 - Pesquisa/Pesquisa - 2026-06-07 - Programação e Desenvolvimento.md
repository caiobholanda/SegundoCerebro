# Programação e Desenvolvimento — 2026-06-07

## Rankings de Linguagens

- **Python**: #1 TIOBE com 21,81% (fev/2026) — queda do pico de 26,98% em julho/2025; posição sustentada pelo ecossistema AI/ML (PyTorch, TF, Hugging Face, LangChain)
- **TypeScript**: #1 GitHub por contribuidores (pela 1ª vez em 2025); crossover de PR volume sobre JavaScript iminente
- **Rust**: 50% das empresas já usam em produção (adoção +10pp em 2 anos); substituindo C e C++ em SO, browsers, DBs e cloud infra

## Tendências de Adoção

- 94% dos erros em código gerado por IA são relacionados a tipo — linguagens com tipagem estática (TypeScript, Rust, Java) capturam automaticamente → crescem junto com AI-assisted dev
- **Tiny Teams** amplificados por IA — indivíduos entregando o que antes exigia equipes
- Dev = 70% revisor e arquiteto; 30% escritor de código

## Desenvolvimento AI-Native

- >84% dos devs usam IA no workflow (Copilot, CodeWhisperer, Claude Code como baseline)
- Engenharia de prompt = competência core em 2026
- **Vibe Coding** popularizando-se para prototipagem rápida

## Padrões de Stack

| Domínio | Stack dominante |
|---|---|
| AI/ML | Python + PyTorch / Hugging Face |
| Frontend | TypeScript + React / Next.js |
| Backend | TypeScript (Node.js) ou Python (FastAPI/Django) |
| Infra crítica / sistemas | Rust substituindo C/C++ |
| Microserviços / cloud-native | Go crescendo |
| Web Assembly | Rust + WASM |

## Pré-requisitos 2026

- **DevSecOps**: segurança integrada ao ciclo de dev (não mais diferencial)
- **Green Software**: eficiência energética como critério de qualidade
