# Pesquisa - Claude Code — CLI e agentes

## ❓ Pergunta central
Como o Claude Code utiliza CLI e agentes para expandir suas capacidades de automação e integração?

## 🎯 Síntese (3-5 linhas)
O Claude Code combina ferramentas de linha de comando (CLI) com agentes inteligentes para criar fluxos de trabalho automatizados e integrados. A CLI possibilita interações diretas entre sistemas, enquanto os agentes são programados para tomar decisões autônomas com base em regras e contexto. Juntos, esses elementos tornam o Claude altamente adaptável para tarefas complexas e personalizadas.

## 🔬 Detalhes
- CLI (Command Line Interface) permite interações diretas e controladas entre o Claude e outros sistemas, sem necessidade de interfaces gráficas.
- Agentes inteligentes são módulos autônomos que processam informações e tomam decisões baseadas em contexto e regras predefinidas.
- Claude utiliza APIs como a [[Claude API e Anthropic SDK]] para conectar CLI e agentes a serviços externos.
- A integração de CLI com agentes facilita tarefas como busca de dados, execução de scripts automatizados e geração de relatórios.
- Agentes podem ser configurados para monitorar eventos, executar tarefas recorrentes ou responder a comandos em tempo real.
- Antropic, a desenvolvedora do Claude, prioriza segurança e ética na IA, garantindo que agentes tenham limitações claras para evitar uso indevido.
