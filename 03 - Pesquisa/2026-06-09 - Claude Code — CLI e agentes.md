# Pesquisa - Claude Code — CLI e agentes

## ❓ Pergunta central
Como o Claude Code pode ser integrado e utilizado via CLI e agentes personalizados?

## 🎯 Síntese (3-5 linhas)
O Claude Code combina o poder de inteligência artificial conversacional com automações e integrações via CLI (Interface de Linha de Comando). Ele permite criar agentes personalizados que executam tarefas específicas com eficiência, aproveitando o modelo da Anthropic. A integração com sistemas legados, a facilidade de uso e a personalização são seus principais diferenciais.

## 🔬 Detalhes
- **Claude Code** é uma extensão do modelo de linguagem Claude, da Anthropic, voltada para automações e interações com sistemas via CLI.
- Ele permite a criação de agentes personalizados que podem ser configurados para realizar tarefas específicas, como manipulação de arquivos, execução de comandos e integração com APIs.
- A interface CLI do Claude Code é projetada para desenvolvedores e equipes técnicas que buscam otimizar processos e automatizar fluxos de trabalho.
- O modelo utiliza uma abordagem baseada em princípios para garantir que os agentes atuem de forma ética e segura, minimizando riscos de mau uso.
- Sua arquitetura é compatível com o **Anthropic SDK**, permitindo integração com outros produtos da Anthropic e linguagens de programação populares como Python e JavaScript.
- Casos de uso incluem automação de tarefas repetitivas, suporte técnico, integração com ferramentas DevOps e até mesmo geração de código ou documentação.
- O modelo suporta o desenvolvimento iterativo, onde os agentes podem ser treinados ou ajustados com feedback contínuo de usuários.
- Recursos avançados incluem suporte para comandos em tempo real, logs detalhados para auditoria e personalização de respostas com base na identidade organizacional.
