# Programação e Desenvolvimento — 2026-06-12

## Rankings de linguagens

### Python — TIOBE #1

- Share: ~23,64% (queda do pico de 26,98%, mas mantém liderança)
- Crescimento de 7pp de 2024 para 2025 — maior salto de qualquer linguagem major
- Dominância absoluta em IA/ML/Data Science — tendência irreversível
- Uso geral: 57,9% dos desenvolvedores (Stack Overflow 2026)

### TypeScript — GitHub #1

- Primeira vez na história ultrapassando Python e JavaScript por número de contribuidores no GitHub
- Crescimento: +66% YoY (desde agosto 2025)
- Padrão de fato em desenvolvimento web profissional
- Uso: 43,6% dos desenvolvedores (Stack Overflow 2026)
- >80% dos novos projetos JavaScript usam TypeScript por padrão

### Rust — mais admirada pelo 9º ano consecutivo

- 72% de taxa de admiração (Stack Overflow 2026)
- Reescrevendo infraestrutura crítica: SOs, browsers, bancos de dados, cloud infra
- +35% de vagas abertas para Rust em 2026
- Presente em produção em ~50% das empresas Fortune 500

## Tendência macro: tipagem estática vence

O mercado se moveu decisivamente para linguagens type-safe:
- TypeScript > JavaScript
- Rust > C/C++
- Kotlin > Java

Motivo: 94% dos erros introduzidos por IA são tipo-relacionados. Com IA gerando mais código, tipagem estática vira rede de segurança essencial.

## Stack moderna 2026

```
Frontend         → TypeScript (React, Next.js, Svelte)
AI/ML services   → Python (FastAPI, PyTorch, LangChain)
High-perf/infra  → Rust (WASM, CLI tools, sistemas)
Cloud/microserv  → Go (APIs, microsserviços)
```

Uma aplicação moderna usa as quatro em camadas diferentes.

## Papel do desenvolvedor em 2026

- **70% do tempo**: revisor, arquiteto, validador de código gerado por IA
- **84%+** dos devs usam IA no dia-a-dia (GitHub Copilot como baseline, não diferencial)
- "Vibe coding" consolidado como prática — preocupação com qualidade e type-safety aumenta em paralelo
- DevSecOps + Green Software: de diferenciais para pré-requisitos de contratação
