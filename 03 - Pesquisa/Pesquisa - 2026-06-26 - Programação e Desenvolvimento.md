# Programação e Desenvolvimento — 2026-06-26

## Ranking de Linguagens

| Ranking | Linguagem | Índice | Destaque |
|---------|-----------|--------|----------|
| #1 TIOBE | Python | abaixo de 19% | IA/ML dominante |
| #1 GitHub | TypeScript | 2,6M+ contribuidores/mês | superou Python em ago/2025 |
| Rising | Rust | 10º ano mais admirada | sistemas críticos |
| Crescendo | Kotlin | preferido sobre Java | mobile/backend |

## Python

- Consolidado como linguagem #1 para IA, ML, ciência de dados e automação
- TIOBE: próximo de 19% — queda leve por diversificação, mas liderança indiscutível
- Maior número de vagas abertas no Brasil em TI

## TypeScript

- #1 GitHub por contribuidores desde agosto 2025 (primeira vez na história)
- Crescimento de 66% ano a ano em 2025
- **80%+ dos novos projetos JavaScript usam TypeScript** em 2026
- Type-safety venceu definitivamente: TS > JS, Kotlin > Java, Rust > C++
- 94% dos erros em código gerado por IA são tipo-relacionados → TS mitiga isso

## Rust

- 10º ano consecutivo como linguagem "mais admirada" (Stack Overflow)
- Adoção em sistemas críticos: Microsoft, Google, Amazon, Meta, Linux kernel
- +35% de vagas abertas ano a ano
- Forte em: cloud-native, segurança, sistemas embarcados, blockchain

## Tendências Macro em 2026

**IA Aplicada ao Dev**
- 84%+ dos desenvolvedores usam IA no trabalho diário
- Dev = 70% revisor/arquiteto de código gerado por IA + 30% codificador
- Vibe coding em ascensão — mas type-safety ainda previne a maioria dos erros

**DevSecOps**
- Segurança integrada desde o início do ciclo de desenvolvimento
- Pré-requisito em 70%+ das vagas sênior

**Green Software / Programação Sustentável**
- Redução de pegada de carbono de software tornando-se critério de contratação
- Algoritmos eficientes > código rápido de escrever

**WebAR e Edge AI**
- WebAR emergindo com 5G para experiências imersivas sem app
- Edge AI: modelos rodando no dispositivo (Apple Foundation Models, Android on-device)

## Kubernetes e Containers

- Kubernetes: requisito em 70%+ das vagas sênior em 2026
- Docker: fundação de qualquer stack de produção
