# Pesquisa - Claude Code — CLI e agentes

## ❓ Pergunta central
Como o Claude Code pode ser utilizado via CLI para aprimorar a interação com agentes e fluxos de trabalho?

## 🎯 Síntese (3-5 linhas)
O Claude Code em CLI oferece uma interface poderosa para interagir com agentes baseados em IA diretamente do terminal, permitindo automações, execução de tarefas específicas e interações dinâmicas. Ele utiliza a infraestrutura da API da Anthropic para fornecer respostas contextuais e executar comandos personalizados. Sua integração com fluxos de trabalho permite aumentar a produtividade e simplificar processos complexos.

## 🔬 Detalhes
- O Claude Code CLI permite que desenvolvedores e usuários avancem na automação de tarefas diretamente de um terminal, sem necessidade de interfaces gráficas.
- A ferramenta utiliza a API da Anthropic, que é baseada em modelos de linguagem avançados para oferecer respostas contextuais e executar comandos.
- Suporte para criação de agentes personalizados que podem ser configurados para resolver problemas ou realizar tarefas específicas.
- Pode ser integrado a pipelines de automação, como CI/CD, para interagir diretamente com sistemas e processos complexos.
- Oferece suporte a scripts customizáveis, permitindo que usuários criem fluxos de trabalho adaptados às suas necessidades.
- Um diferencial do Claude Code CLI é a capacidade de manter um "contexto ativo", lembrando de informações relevantes durante a sessão de uso.
- Suporte a múltiplos idiomas, permitindo que desenvolvedores de diferentes regiões possam utilizar a CLI em seus idiomas nativos.
- Documentação robusta e código aberto para facilitar a personalização, aprendizado e colaboração da comunidade técnica.
