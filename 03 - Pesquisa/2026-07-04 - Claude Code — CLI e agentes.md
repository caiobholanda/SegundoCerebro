# Pesquisa - Claude Code — CLI e agentes

## ❓ Pergunta central
Como a interface CLI e os agentes do Claude Code podem ser utilizados para automação e integração em fluxos de trabalho?

## 🎯 Síntese (3-5 linhas)
O Claude Code oferece uma CLI poderosa para interagir com o modelo de linguagem da Anthropic, permitindo integração com diversos sistemas e automação de tarefas complexas. Os agentes são uma extensão que possibilita fluxos de trabalho personalizados, com lógica programável e interações autônomas. Ambas ferramentas são úteis para desenvolvedores e equipes que desejam construir soluções baseadas em IA de forma eficiente e robusta.

## 🔬 Detalhes
- O Claude Code CLI serve como uma interface de linha de comando para interagir diretamente com os modelos, permitindo automação e integração com scripts e outros sistemas.
- Os agentes Claude são estruturas programáveis que atuam como intermediários inteligentes, capazes de tomar decisões e executar fluxos de trabalho.
- A CLI suporta comandos para configuração, execução de prompts, modificação de parâmetros de geração e consulta de histórico de interações.
- Os agentes podem ser configurados com regras específicas para lidar com diferentes tipos de entradas e produzir saídas personalizadas.
- Existe suporte para integração com APIs externas, permitindo que dados de diferentes fontes sejam processados e utilizados nas interações.
- O uso de Claude Code CLI e agentes é ideal para tarefas como análise de dados, geração de relatórios, atendimento automatizado e suporte técnico.
- Segurança e controle são pontos focais da Anthropic, com recursos para limitar o acesso e monitorar as interações dos agentes.
