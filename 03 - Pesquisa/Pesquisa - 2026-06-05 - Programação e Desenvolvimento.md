# Programação e Desenvolvimento — 2026-06-05

## Destaque principal

**Linguagens type-safe dominam**: a indústria se moveu decisivamente para TypeScript > JavaScript, Rust > C/C++, Kotlin > Java — erros capturados em compile time em vez de runtime. TypeScript é agora #1 no GitHub por contribuidores (desde agosto 2025).

## Estado atual das linguagens (2026)

### Python
- **#1 TIOBE**: ~23,64% de market share (dezembro 2025, sustentado em 2026)
- Posição deve-se quase inteiramente ao ecossistema de IA/ML: PyTorch, TensorFlow, Hugging Face, LangChain
- **Desafio emergente**: para inference em produção, empresas usam Rust, C++ ou runtimes especializados em vez de Python puro
- **Python + Rust via PyO3/maturin**: padrão para acelerar hot paths sem reescrever o codebase inteiro

### TypeScript
- **#1 GitHub por contribuidores** (1ª vez, desde ago/2025 — +66% YoY, +1M novos devs em 2025)
- Demand para devs TypeScript em máxima histórica em 2026
- **>80% dos novos projetos JavaScript** usam TypeScript como padrão
- Codebases tipados escalam melhor, onboardam mais rápido e integram mais seguramente com ferramentas de desenvolvimento IA

### Rust
- **8º ano consecutivo como linguagem mais admirada** (Stack Overflow 2026)
- Está substituindo C e C++ em **infraestrutura crítica**: SO, browsers, bancos de dados, cloud infrastructure
- **+35% nas vagas** em 2026
- Propriedades de memory safety, alta performance e segurança robusta o tornam ideal para sistemas críticos
- PyO3/maturin: bridge Python↔Rust para hot paths

## Tendências transversais

- **AI-Native Development**: >84% dos devs usam IA; papel do dev = 70% revisor/arquiteto em vez de escritor de código
- **Type-safe everywhere**: TypeScript, Rust, Kotlin — industry decidiu que erros em compile time valem o custo
- **DevSecOps como pré-requisito**: não mais diferencial — exigido em processos seletivos
- **Green Software**: eficiência energética como métrica de qualidade emergindo em 2026
- **WASM + Rust**: crescendo para aplicações web de alta performance

## O que mudou desde 2026-06-03

- Benchmark Stack Overflow 2026 confirma Rust como 8º ano mais admirada
- Python TIOBE mantém ~23,64% — posição sólida mas desafiada no runtime por Rust/C++
- TypeScript supera Python/JS no GitHub em contribuidores — consolidado como #1
