# Pesquisa - Claude Code — CLI e agentes

## ❓ Pergunta central
Como o Claude Code pode ser utilizado via CLI e quais são as melhores práticas para integrar agentes em fluxos de trabalho automatizados?

## 🎯 Síntese (3-5 linhas)
O Claude Code oferece suporte para integração com sistemas e workflows através de CLI e agentes programáveis. Essa funcionalidade permite executar tarefas automatizadas, realizar análises e integrar com APIs externas. As melhores práticas envolvem o uso de prompts bem definidos, segurança no manuseio de tokens e logs, além de planejamento detalhado das interações.

## 🔬 Detalhes
- O CLI do Claude Code permite interagir com o modelo de linguagem via terminal, possibilitando automação de tarefas repetitivas.
- A integração com agentes é viabilizada por meio de APIs, como a [[Claude API e Anthropic SDK]], que suportam a comunicação com o modelo em diferentes linguagens de programação.
- Um uso comum de agentes é a execução de fluxos de trabalho em ferramentas como Zapier, Make ou scripts customizados, permitindo desencadear ações baseadas nas respostas do modelo.
- Prompts bem estruturados são cruciais para garantir precisão e relevância nas interações com o Claude Code.
- A segurança é um ponto crítico ao trabalhar com agentes: tokens de API devem ser protegidos e logs de conversação precisam ser tratados com cuidado para evitar vazamento de informações sensíveis.
- O uso de agentes em CLI também permite a integração com sistemas legados, ampliando a utilidade do modelo em contextos empresariais.
