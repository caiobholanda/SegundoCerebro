# Programação e Desenvolvimento — 2026-06-24

## Linguagens mais Populares

### Python
- **#1 TIOBE** (21,81% fev/2026) — mantém liderança global
- Dominante em IA/ML, ciência de dados e automação
- Principal linguagem de entrada no mercado de IA

### TypeScript
- **#1 GitHub por contribuidores** (desde ago/2025, mantém)
- Crescimento: +1M novos colaboradores em 2025
- 80%+ dos projetos JavaScript modernos usam TS
- Impulsionado por Next.js, Angular e outros frameworks que adotaram TS como padrão

### JavaScript
- Ainda dominante no front-end; TS o está substituindo no desenvolvimento escalável

### Rust
- **8º/9º ano consecutivo como linguagem mais amada** pelos desenvolvedores
- Adotado por Microsoft, Google, Amazon e Meta para sistemas críticos de performance
- +35% crescimento de vagas abertas
- 50% das Fortune 500 com algum uso em produção

## Tendências de Desenvolvimento

### IA Integrada ao Workflow
- 84%+ dos devs usando ou planejando usar ferramentas com IA em 2026
- Dev = 70% revisor/arquiteto (tendência consolidada desde 2025)
- 94% dos erros introduzidos por IA são tipo-relacionados → justifica TypeScript e tipagem estática

### Type-Safety como Padrão
- TypeScript > JavaScript
- Rust > C++
- Kotlin > Java
- Tipo-safety venceu como abordagem padrão para sistemas críticos

### DevSecOps + Green Software
- Segurança integrada ao pipeline como pré-requisito (não opcional)
- Green Software: eficiência energética do código como métrica de qualidade emergente

### AI-Native Development
- Contexto de 1M tokens (Fable 5) muda workflows de coding — projetos inteiros cabem no contexto
- Código gerado por IA revisado por humanos como fluxo padrão
