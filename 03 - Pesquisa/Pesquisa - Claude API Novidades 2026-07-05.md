# Pesquisa - Claude API Novidades 2026-07-05

## ❓ Pergunta central
O que há de novo na API da Anthropic, modelos e features em julho de 2026?

## 🎯 Síntese
Claude Sonnet 5 é o grande lançamento: 1M tokens de contexto, 128k output, preço introdutório de $2/$10 por MTok até 31/ago. Claude Fable 5 voltou globalmente em 1º de julho após levantamento de restrições de exportação pelo US DoC. Managed Agents agora suportam cron jobs nativos e variáveis de ambiente no vault. Microsoft Foundry com Claude é GA.

## 🔬 Detalhes
- **Claude Sonnet 5**: padrão em planos Free e Pro; $2/MTok input, $10/MTok output (introductory até 31/08); 1M context window; 128k max output
- **Fable 5**: disponível globalmente desde 01/07/2026 após US DoC levantar export controls impostas em 12/06
- **Managed Agents Scheduling**: cron-based automation, vault env vars, integrações com browser
- **Enterprise Analytics**: admin analytics enriquecido, model-level entitlements, spend alerts
- **Microsoft Foundry**: GA com opções Azure-hosted e Anthropic-hosted
- **Rate Limits API**: admins podem consultar rate limits programaticamente por organização/workspace
