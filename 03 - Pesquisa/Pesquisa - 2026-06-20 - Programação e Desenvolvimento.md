# Programação e Desenvolvimento — 2026-06-20

## Rankings de Linguagens (junho 2026)

| Linguagem | Ranking | Destaque |
|-----------|---------|----------|
| Python | #1 TIOBE (~23,64%) | IA/ML/Data Science |
| TypeScript | #1 GitHub (contribuidores) | >80% novos projetos JS |
| Rust | 8º ano mais amado (Stack Overflow) | +35% vagas, memória segura |
| Go | Crescimento cloud-native | Microserviços, K8s |
| Java | Mantém enterprise | Spring, Kotlin substituindo |

## Tendências Principais 2026

### AI-Native Development
- 84%+ dos times de desenvolvimento usam IA em alguma etapa
- GitHub Copilot e Cursor deixaram de ser diferencial — são baseline
- Dev = **70% revisor/arquiteto**, 30% escritor de código novo
- Fable 5 com 1M ctx: mudança em workflows de refactoring de bases grandes

### TypeScript
- +80% dos novos projetos JavaScript usam TypeScript
- 94% dos erros gerados por IA são tipo-relacionados → tipagem estática vence
- Frameworks maiores (React, Next.js, NestJS) usam TS por padrão

### Rust
- 8º ano consecutivo como linguagem mais admirada no Stack Overflow
- 50% das empresas Fortune 500 com Rust em produção
- Cresce em WASM, segurança de sistemas e blockchain
- AI tools (GitHub Copilot, Cursor) reduzem barreira de entrada para Rust

### DevSecOps e Green Software
- DevSecOps: de diferencial → **pré-requisito** em 2026
- Green Software: eficiência de código como métrica de engenharia
  - Redução de consumo energético como KPI
  - Pressão ESG para empresas reportarem emissões de infraestrutura TI

## Papel do Desenvolvedor

- Shift de "escrever código" para "revisar, arquitetar, validar"
- Tiny Teams: IA amplifica 1 dev ao nível de equipes maiores
- Engenharia de prompt como competência técnica core (não soft skill)
- Context Engineering = nova especialidade crítica
