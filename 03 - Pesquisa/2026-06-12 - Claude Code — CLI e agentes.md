# Pesquisa - Claude Code — CLI e agentes

## ❓ Pergunta central
Como o Claude Code pode ser usado via CLI para criar e gerenciar agentes de forma eficiente?

## 🎯 Síntese (3-5 linhas)
O Claude Code oferece uma interface de linha de comando (CLI) robusta para interagir com agentes de IA, permitindo automação e integração com fluxos de trabalho personalizados. Ele facilita a criação, treinamento e gerenciamento de agentes por meio de comandos simples, integrando-se ao Anthropic SDK e outras ferramentas. Apesar de poderoso, ainda há espaço para melhorias em termos de documentação e suporte a fluxos complexos.

## 🔬 Detalhes
- O CLI do Claude Code permite criar e gerenciar agentes diretamente do terminal, facilitando a automação de tarefas.
- Ele se integra ao [[Claude API e Anthropic SDK]], permitindo a criação de pipelines personalizados para treinamento e execução de modelos de linguagem.
- Oferece comandos para gerenciar estados de agentes, como inicialização, pausa, reconfiguração e monitoramento de logs.
- Suporte a scripts e workflows permite integrar agentes a ferramentas externas, como CRMs, softwares de automação e APIs de terceiros.
- A documentação oficial ainda está em desenvolvimento, o que pode dificultar o uso para iniciantes ou usuários sem experiência em CLI.
- Recursos de debugging no CLI são limitados, o que exige maior atenção ao configurar agentes complexos ou fluxos personalizados.
