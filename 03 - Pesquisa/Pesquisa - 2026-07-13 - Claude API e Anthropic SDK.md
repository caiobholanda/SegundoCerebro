# Pesquisa - Claude API e Anthropic SDK — 2026-07-13

## ❓ Pergunta central
Quais são as novidades da API da Anthropic e modelos Claude em julho de 2026?

## 🎯 Síntese
Claude Sonnet 5 tornou-se modelo padrão (Free/Pro) em 30 de junho. Claude Fable 5 é o modelo de fronteira disponível amplamente. Claude Mythos 5 existe mas em acesso limitado (Project Glasswing). Novo preço introdutório de $2/$10 por 1M tokens válido até 31/08/2026.

## 🔬 Detalhes

### Hierarquia de Modelos (Julho 2026)
| Modelo | Disponibilidade | Input | Output |
|--------|----------------|-------|--------|
| Claude Mythos 5 | Limitada (Project Glasswing) | — | — |
| Claude Fable 5 | Geral | — | — |
| Claude Sonnet 5 | Padrão Free/Pro (desde 30/06) | $2/1M | $10/1M |

**Preço padrão após 31/08/2026:** $3/$15 por 1M tokens (entrada/saída).

### Claude Cowork
- Expandido para **web e mobile** (era apenas desktop)
- Sessões e arquivos sincronizados entre dispositivos
- Chat e Cowork agora compartilham o mesmo espaço

### Microsoft 365 Integration
- Rascunho e envio de email pelo Claude
- Gerenciamento de eventos de calendário
- Criar/atualizar arquivos OneDrive e SharePoint

### Analytics para Admins
- Painel mostra uso e custo **por grupo e por usuário**
- Artefatos criados, arquivos editados, skills e conectores junto aos custos

### Reflection Feature (Beta)
- **Settings > Reflect**: resumo mensal de tópicos, dias ativos, padrões de trabalho
- **Settings > Time and focus**: lembretes de intervalo e horários silenciosos
- Disponível para planos Free, Pro e Max
