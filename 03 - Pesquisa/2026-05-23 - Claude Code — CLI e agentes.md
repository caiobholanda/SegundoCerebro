# Pesquisa - Claude Code — CLI e agentes

## ❓ Pergunta central
Como o Claude Code pode ser utilizado via CLI e agentes para potencializar fluxos de trabalho de IA?

## 🎯 Síntese (3-5 linhas)
O Claude Code oferece suporte a interações com IA por meio de uma interface de linha de comando (CLI), permitindo automação e integração com sistemas existentes. Além disso, sua funcionalidade de agentes possibilita o desenvolvimento de fluxos de trabalho personalizados que ampliam as capacidades de produtividade e eficiência. A documentação detalhada e o suporte a múltiplas linguagens de programação tornam o Claude Code uma ferramenta versátil para desenvolvedores.

## 🔬 Detalhes
- O Claude Code CLI permite que usuários interajam diretamente com o modelo de IA através de comandos no terminal, facilitando a execução de tarefas automatizadas.
- Pode ser integrado com ferramentas de desenvolvimento como scripts em Python, pipelines de CI/CD e automações com shell scripts.
- A funcionalidade de agentes no Claude Code permite a criação de fluxos de trabalho automatizados, onde a IA pode executar tarefas específicas com base em regras predefinidas.
- Os agentes são configuráveis por meio de arquivos JSON ou YAML, permitindo personalização para diferentes casos de uso.
- O suporte a múltiplas linguagens de programação, como Python, JavaScript e Go, amplia as possibilidades de uso para desenvolvedores de diferentes áreas.
- A robustez da CLI facilita a integração com APIs existentes, como [[Claude API e Anthropic SDK]], para construir soluções completas e interativas.
- A documentação oficial detalha exemplos práticos, como o uso de agentes para geração de texto, resumo de documentos e automação de respostas de e-mail.
