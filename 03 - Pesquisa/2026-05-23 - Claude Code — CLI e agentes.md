# Pesquisa - Claude Code — CLI e agentes

## ❓ Pergunta central
Como o Claude Code CLI e agentes podem ser utilizados para automação avançada de tarefas e integração em fluxos de trabalho?

## 🎯 Síntese (3-5 linhas)
Claude Code CLI e seus agentes oferecem uma interface poderosa para interagir com o modelo de linguagem da Anthropic. Permitem a execução de tarefas programáticas, integração com sistemas externos e construção de fluxos de trabalho personalizados. Entender suas capacidades e limitações é essencial para aproveitar ao máximo sua aplicação em desenvolvimento e automação.

## 🔬 Detalhes
- O **Claude Code CLI** é uma ferramenta de linha de comando que facilita interações diretas com o modelo Claude, permitindo automação e prototipagem rápida.
- **Agentes Claude** podem ser configurados para executar tarefas específicas, como buscar informações, processar texto ou integrar APIs externas.
- A CLI suporta diferentes configurações de modelos, como Claude 2 e Claude Instant, com variações em desempenho e custo.
- A integração com **Anthropic SDK** amplia as capacidades do CLI, oferecendo maior controle e personalização.
- Claude Code CLI é projetado para ser utilizado em sistemas operacionais baseados em UNIX, com suporte para scripts em Shell e Python.
- Segurança e privacidade são prioridades no CLI, com mecanismos para gerenciar tokens de API e criptografia de dados sensíveis.
