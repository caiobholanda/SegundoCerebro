# Claude Code — CLI e Agentes — 2026-06-16

## Agent View

- Gerenciamento de múltiplas sessões a partir de uma única interface CLI
- Operações disponíveis: iniciar agentes, enviar para background, checar status e respostas, retomar sessões quando precisam de input

## Dynamic Workflows

- Disponível em Research Preview para planos Max, Team e Enterprise
- Permite resolver problemas complexos e em grande escala via CLI, Desktop e VS Code extension
- Orquestra dezenas a centenas de subagentes em paralelo com verificação embutida

## Melhorias de Permissões e Segurança

- Regras de permissão mais granulares com sintaxe `Tool(param:value)` — permite matching de parâmetros de input com wildcards
- Suporte a nested `.claude` — skills em `.claude/skills` em subdiretórios carregam automaticamente ao trabalhar em arquivos daquele diretório
- Auto mode mais seguro

## Microsoft → Copilot CLI (30/06/2026)

- Microsoft planeja migrar licenças corporativas do Claude Code para Copilot CLI em **30 de junho de 2026**
- Empresas que usam Claude Code via licença Microsoft devem estar cientes do prazo

## Tendência Macro

- Claude Code agora representa uma mudança de software como ferramenta para software como trabalhador semi-autônomo
- Shift fundamental: de assistente de código para agente de engenharia independente

## Correções e Confiabilidade

- Fixes para crashes, problemas de autenticação, websocket, confiabilidade de subagentes
- Melhorias diversas de interface
