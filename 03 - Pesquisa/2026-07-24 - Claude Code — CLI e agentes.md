# Pesquisa - Claude Code — CLI e agentes

## ❓ Pergunta central
Como o Claude Code pode ser utilizado via CLI e integrado a agentes para automação de tarefas?

## 🎯 Síntese (3-5 linhas)
O Claude Code, desenvolvido pela Anthropic, oferece ferramentas para interação com modelos de linguagem via CLI, permitindo a integração com agentes automatizados. Sua arquitetura é projetada para simplificar fluxos de trabalho e habilitar automações personalizáveis. Com suporte avançado a prompts e APIs, ele possibilita a orquestração eficiente de processos complexos em várias linguagens de programação.

## 🔬 Detalhes
- O CLI do Claude Code é uma interface que permite executar prompts diretamente do terminal, útil para desenvolvedores e automações.
- Suporta integração com ferramentas de orquestração de tarefas, como Airflow, Zapier e Make.com, para fluxos de trabalho mais complexos.
- APIs da Anthropic permitem que o Claude Code seja usado como backend em agentes personalizados, fornecendo respostas contextuais e execução de comandos.
- A plataforma inclui suporte a versões específicas do modelo Claude, ajustáveis conforme a necessidade de custo ou desempenho.
- Agentes podem ser configurados para executar tarefas como geração de código, análise de dados e manipulação de arquivos, utilizando as habilidades do modelo.
- Há uma crescente adoção do Claude Code como backend em projetos que requerem LLMs para automação em tempo real.
